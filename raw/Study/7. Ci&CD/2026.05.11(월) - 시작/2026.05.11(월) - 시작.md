2) ci.yml 파일 작성


workflows 디렉토리 우클릭 후 file 생성 (파일명 : ci.yml)
(교육 자료 - A02.설정용 파일 - ci_first.yml 코드 복붙)

name: Spring Boot CI

on:
  push:
    branches:
      - master

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout source
        uses: actions/checkout@v4

      - name: Set up JDK 21
        uses: actions/setup-java@v4
        with:
          java-version: '21'
          distribution: 'temurin'
          cache: maven

      - name: Build with Maven
        run: mvn clean package -DskipTests

-> master 브랜치로 push를 (on)하면 JDK 21이고 Java 버전이 21인 상태로 maven 형식으로 패키지를 빌드한다.

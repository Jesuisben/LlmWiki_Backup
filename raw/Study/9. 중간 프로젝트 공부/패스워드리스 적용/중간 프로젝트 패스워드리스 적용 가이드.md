##### 1. 도커 네트워크 생성
1) 개념
도커 컨테이너는 각각 격리된 작은 가상 컴퓨터처럼 동작함
기본적으로 서로 분리되어 있어서 서로 다른 컨테이너를 직접 부르기 어려움
이때 도커 네트워크를 만들어서 컨테이너를 이 네트워크에 연결해놓으면
같은 네트워크에 속한 컨테이너끼리는 이름만으로 서로를 찾아서 통신이 가능해짐
패스워드리스는 컨테이너가 1개여도 그 컨테이너 안에 서버3개 + 내장 DB가 있어서
지금 당장 도커 네트워크는 필요없지만 표준 설치 절차가 이렇고 나중에 확장을 쉽게 하기 위해서 설치함

2) 실습

(설명) 패스워드리스 컨테이너 실행 명령어 한 줄씩 의미

```shell
sudo docker run -d --name server \
```
  → docker run : 컨테이너를 새로 만들어서 실행해라
  → -d : 백그라운드(detached)로 실행 (터미널 안 잡아먹고 뒤에서 조용히 돌게)
  → --name server : 이 컨테이너의 이름을 'server'로 지정 (나중에 docker ps/stop 등에서 이 이름으로 부름)

```shell
--network some-network \
```
  → 위에서 만든 'some-network' 가상 통신망에 이 컨테이너를 연결


```shell
--restart always \
```
  → 컨테이너가 꺼지거나 서버(EC2)가 재부팅돼도 자동으로 다시 켜라


```shell
--cpus=1 \
```
  → 이 컨테이너가 쓸 수 있는 CPU를 최대 1코어로 제한


```shell
--cpuset-cpus="0" \
```
  → 여러 CPU 코어 중 0번 코어만 사용하도록 고정 (특정 코어에 묶어둠)


```shell
--cpu-shares=1024 \
```
  → CPU 경쟁이 생겼을 때의 우선순위 가중치 (1024는 기본값. 다른 컨테이너와 비교한 상대적 몫)


```shell
--memory=1g \
```
  → 이 컨테이너가 쓸 수 있는 RAM(물리 메모리)을 최대 1GB로 제한


```shell
--memory-swap=2g \
```
  → 메모리+스왑 합쳐서 최대 2GB까지 허용 (물리 1GB 다 쓰면 나머지 1GB는 디스크 스왑으로)

```java
package com.backend_semi.passwordless.config;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.reactive.function.client.WebClient;

@Configuration
public class PasswordlessClientConfig {

    // 인증서버 호출 전용 WebClient
    @Bean
    // application.properties파일에 있는 데이터를 변수로 만들어서 넣은 PasswordlessProperties.java에 있는
    // 변수들을 매개변수로 사용하는데
    // 거기에 있는 패스워드리스 인증 서버인 authServerUrl를 baseUrl로 사용해서
    // 나중에 Controller에서 매핑을 할때 사용하는 주소에 하나하나 auth.jesuisben.store:11040를
    // 넣을 필요없이 바로 그 뒤의 주소를 넣을 수 있게 사용함
    // 프론트의 axiosInstance에 axios.create같은 것
    // 나중에 이 도구를 패스워드리스를 호출할때 사용함
    public WebClient passwordlessWebClient(PasswordlessProperties props){
        return WebClient.builder()
                .baseUrl(props.getAuthServerUrl())   // http://auth.jesuisben.store:11040
                .build();
    }
}
```

여러개의 DTO를 만들지 않고 한곳에서 관리하게 해놓음
-> 아예 틀 자체가 틀린 응답도 있어서 그 응답들은 별도의 DTO를 만들 예정

** 해당 패스워드리스 서버에 요청해서 받을 응답이 어떻게 구성되는지는 Postman으로 응답을 받아보거나
** 명세서를 보는수밖에 없음
** 
(passwordless/dto 폴더 생성/PwlessResponse.java 파일 생성)

3-3. 요청을 보내고 응답을 받는 Client 파일 생성
요청을 받고 응답을 보내는 Controller 파일이 있다면
요청을 보내고 응답을 받는 Client 파일이 있음
-> Controller의 반대 버전이라고 생각하면 좋음
(passwordless/client 폴더 생성/PasswordlessClient.java 파일 생성 )

- 코드 작성
```java
package com.backend_semi.passwordless.client;

import com.backend_semi.passwordless.config.PasswordlessProperties;
import com.backend_semi.passwordless.dto.PwlessResponse;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Component;
import org.springframework.util.LinkedMultiValueMap;
import org.springframework.util.MultiValueMap;
import org.springframework.web.reactive.function.BodyInserters;
import org.springframework.web.reactive.function.client.WebClient;

import javax.crypto.Cipher;
import javax.crypto.spec.IvParameterSpec;
import javax.crypto.spec.SecretKeySpec;
import java.nio.charset.StandardCharsets;
import java.util.Base64;

@Component // 스프링에서 관리하는 부품으로 등록하는 어노테이션 -> 그래야 Service에서 사용할 수 있음
@RequiredArgsConstructor
public class PasswordlessClient {

    // PasswordlessClientConfig에서 만든 요청용 WebClient 사용하기 (프론트의 axios 느낌)
    private final WebClient passwordlessWebClient;
    // application.properties에 있는 데이터를 사용하기 위해 설정
    private final PasswordlessProperties props;

    // 공통 POST 호출 (form-urlencoded) (7개의 함수에 총 7번 사용됨)
    // 모든 API 호출은 공통된 uri를 가지고 있어서 그 부분을 설정
    // 보낼 데이터를 form에 넣음
    // 반환은 PwlessResponse
    // MultiValueMap 컬렉션 : 원래 Map 컬렉션은 키 1개가 값 1개와 대응하는데
    // 이 컬렉션은 1개의 키에 여러개의 값이 대응되게 할 수 있음 (값은 List 컬렉션으로 들어가짐)
    // 사실상 Map<String, List<String>> 이 구조라고 보면 됨
    private PwlessResponse post(String path, MultiValueMap<String, String> form){
        // 모든 요청이 post 요청임
        return passwordlessWebClient.post()
                .uri("/ap/rest/auth/" + path)
                // 내가 보낼 데이터의 폼 형식 (Postman에서의 x-www-form-urlencoded)
                .header("Content-Type", "application/x-www-form-urlencoded")
                // 보낼 데이터인 form을 요청 body에 넣음
                .body(BodyInserters.fromFormData(form))
                // 요청을 보내고 응답을 받아오라는 명령어
                .retrieve()
                // 응답받을 JSON 형태의 데이터를 어떤 타입의 객체로 변환할지 정함
                // ()안에 적은 타입으로 응답받을 JSON 형태의 데이터가 변환됨
                // ()안에 적은 타입을 함수의 반환타입에도 맞춰서 적어야 함
                .bodyToMono(PwlessResponse.class)
                // 동기 호출 -> 비동기 호출의 반대 - 응답이 올때까지 기다리라는 설정
                // (axios의 await 느낌)
                .block();   // 동기 호출
    }

    // --- 2.1 isAp : 등록 여부 확인 ---
    // 매개변수에는 요청시 body에 보낼 데이터 넣음
    // 요청 매개변수 (userId)
    // 응답 데이터 (result, msg, code, data(exist))
    public boolean isRegistered(String userId){
        // MultiValueMap의 구현체 생성으로 인해 form 객체 생성
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        // MultiValueMap 컬렉션 안에 "userId"키에 userId 값을 넣음
        form.add("userId", userId);
        // 위에서 만든 post함수의 타입이 PwlessResponse 여서 이렇게 사용
        // post("isAp", form) 이것은 저 함수를 하고 반환받은 passwordlessWebClient.post()의 값이 들어있음
        // 따라서 PwlessResponse res에는 post("isAp", form) API 요청을 하고 받은 데이터가 있음
        // 프론트의 response와 같은 느낌
        // return을 boolean값으로 해야해서 굳이 DTO 객체를 생성해서 넣음
        PwlessResponse res = post("isAp", form);
        // 명세서에 적힌 응답 데이터들로 등록여부를 판단함 (result, msg, code, data(exist))
        // 반환하는데 이 많은 항목중에서 하나라도 실패하면 false 반환
        // 다 통과하면 true 반환하면서 등록이 되었다는 걸 보여줌
        return res != null && res.isResult() // boolean 값은 get이 아니고 is를 사용해서 가져옴
                && res.getData() != null
                // 응답 받은 데이터 중에서 중요한 data키 안에 있는 키인 exist를 찾아서 그 키의 값이 false인지 확인
                // 그 값이 false면 등록이 안된거임 (true면 등록이 된거임)
                // data의 타입이 JsonNode여서 새로운 문법인 path("")나 asBoolean()같은게 사용됨
                // JsonNode 타입인 data의 안에 있는 exist의 값은 JsonNode 타입이여서 이걸 스프링(자바)에서
                // 사용하려고 변환하는 함수가 asBoolean() 함수임
                && res.getData().path("exist").asBoolean(false);
    }

    // --- 2.2 joinAp : 등록 정보(QR 등) 요청 ---
    // 요청 매개변수 (userId, name, email)
    // 응답 데이터 (result, msg, code, data(qr, corpId, registerKey, terms, serverUrl,
    // pushConnectorUrl, pushConnectorToken, userId))
    public PwlessResponse joinAp(String userId, String name, String email){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("userId", userId);
        // 굳이 if (name != null) 조건식을 넣은 이유 : 명세서를 보면 userId는 Nullable이 N이고
        // name과 email은 Nullable이 Y여서 이 2개의 값은 없어도 되지만
        // userId가 있을때만 요청을 보내라는 의미이기에 조건식을 설정함
        if (name != null)  form.add("name", name);
        if (email != null) form.add("email", email);
        return post("joinAp", form);
    }

    // --- 2.3 getTokenForOneTime : 암호화 토큰 요청 → 복호화해서 반환 ---
    // 2.4 getSp에 사용될 일회용 토큰
    // 요청 매개변수 (userId)
    // 응답 데이터 (result, msg, code, data(token))
    public String getDecryptedToken(String userId){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("userId", userId);
        // 이 함수의 반환 타입이 String이여서 객체를 생성해야 함
        PwlessResponse res = post("getTokenForOneTime", form);
        if (res == null || !res.isResult() || res.getData() == null) {
            throw new IllegalStateException("일회용 토큰 요청 실패");
        }
        // JsonNode 타입인 token의 값을 자바에서 쓸 수 있게 String으로 변환함
        String encrypted = res.getData().path("token").asText();
        // 변환해서 String이 된 토큰 값을 복호화해서(암호화를 풀어서) 반환함
        return decryptAes(encrypted);   // serverKey로 AES-128-CBC 복호화
    }

    // --- 2.4 getSp : 자동패스워드 생성(인증 시작) ---
    // servicePassword : 인증에 필요한 6자리 숫자
    // 요청 매개변수 (token, userId, random, sessionId, clientIp)
    // 응답 데이터
    // - 200 OK : (result, msg, code, data(term, pushConnectorUrl, pushConnectorToken,
    // servicePassword, userId))
    public PwlessResponse getSp(String token, String userId, String random,
                                String sessionId, String clientIp){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("token", token);
        form.add("userId", userId);
        form.add("random", random);
        form.add("sessionId", sessionId);
        // 명세서에 clientIp는 Nullable이 Y여서 조건식을 사용
        if (clientIp != null) form.add("clientIp", clientIp);
        return post("getSp", form);
    }

    // --- 2.5 result : 모바일 승인 여부 폴링(결과 정보를 계속 요청함) ---
    // 요청 매개변수 (userId, sessionId)
    // 응답 데이터
    // - 200, 요청없음 or cancel api 호출 : (result, msg, code)
    // - 200, W (모바일 대기) : (result, msg, code, data(auth, userId, hash))
    // - 200, N (모바일 거절) : (result, msg, code, data(auth, userId, hash))
    // - 200, Y (모바일 승인) : (result, msg, code, data(auth, userId, hash))
    public PwlessResponse result(String userId, String sessionId){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("userId", userId);
        form.add("sessionId", sessionId);
        return post("result", form);
    }

    // --- 2.6 cancel : 인증 취소 ---
    // 요청 매개변수 (userId, sessionId)
    // 응답 데이터 (result, msg, code)
    public PwlessResponse cancel(String userId, String sessionId){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("userId", userId);
        form.add("sessionId", sessionId);
        return post("cancel", form);
    }

    // --- 2.7 withdrawalAp : 해지 ---
    // 요청 매개변수 (userId)
    // 응답 데이터 (result, msg, code)
    public boolean withdrawal(String userId){
        MultiValueMap<String, String> form = new LinkedMultiValueMap<>();
        form.add("userId", userId);
        PwlessResponse res = post("withdrawalAp", form);
        return res != null && res.isResult();
    }

    // ===== AES-128-CBC 복호화 (key == iv, serverKey 16바이트 사용) =====
    // Postman 스크립트가 Web Crypto로 하던 것을 Java로 옮긴 것.
    private String decryptAes(String base64Cipher){
        try {
            byte[] keyBytes = props.getServerKey().getBytes(StandardCharsets.UTF_8);
            // 명세: key와 iv가 동일 (serverKey 그대로 사용, 16바이트여야 함)
            SecretKeySpec keySpec = new SecretKeySpec(keyBytes, "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(keyBytes);

            Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");
            cipher.init(Cipher.DECRYPT_MODE, keySpec, ivSpec);

            // URL-safe base64 보정 후 디코딩
            String fixed = base64Cipher.replace('-', '+').replace('_', '/');
            byte[] decoded = Base64.getDecoder().decode(fixed);
            byte[] plain = cipher.doFinal(decoded);
            return new String(plain, StandardCharsets.UTF_8);
        } catch (Exception e) {
            throw new IllegalStateException("일회용 토큰 복호화 실패", e);
        }
    }
}
```

#### \# 5. PasswordlessService (등록/로그인/해지 흐름 조립)
(passwordless/service 폴더 생성/PasswordlessService.java 파일 생성)

- 코드 작성
```java
package com.backend_semi.passwordless.service;

import com.backend_semi.entity.Member;
import com.backend_semi.passwordless.client.PasswordlessClient;
import com.backend_semi.passwordless.dto.*;
import com.backend_semi.passwordless.frontDto.AuthResultDto;
import com.backend_semi.passwordless.frontDto.AuthStartDto;
import com.backend_semi.passwordless.frontDto.RegisterInfoDto;
import com.backend_semi.repository.MemberRepository;
import com.backend_semi.security.JwtUtil;
import com.fasterxml.jackson.databind.JsonNode;
import lombok.RequiredArgsConstructor;
import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.UUID;

@Service
@RequiredArgsConstructor
public class PasswordlessService {

    private final PasswordlessClient client;
    private final MemberRepository memberRepository;
    private final JwtUtil jwtUtil;   // 기존 JWT 발급기 재사용
    // 패스워드리스 등록/해지할때 비밀번호를 필요로 함
    private final PasswordEncoder passwordEncoder; // 기존 비번 검증기 재사용 (BCrypt)

    // 패스워드리스 등록/해지할때 비밀번호를 필요로 함
    // ===== 공통: 비밀번호로 본인 확인 후 회원 반환 =====
    // 등록/해지는 로그인 없이 호출되므로, 여기서 ID+PW로 본인 확인을 한다.
    // (기존 MemberService.login과 동일한 검증 방식)
    private Member verifyMember(String loginId, String loginPassword){
        Member member = memberRepository.findByLoginId(loginId)
                .orElseThrow(() -> new IllegalArgumentException("존재하지 않는 아이디입니다."));
        // 사용자가 입력한 ID에 맞지 않는 비밀번호를 입력시
        if (loginPassword == null
                || !passwordEncoder.matches(loginPassword, member.getPassword())) {
            throw new IllegalArgumentException("비밀번호가 일치하지 않습니다.");
        }
        // 사용자가 입력한 ID에 맞는 회원이 있고 그 회원의 비밀번호가 입력한 비밀번호와 일치하면 통과
        return member;
    }

    // ===== 등록: QR 정보 발급 =====
    @Transactional(readOnly = true) // DB를 읽기만하고 안바꾸니까 readOnly(회원조회)
    public RegisterInfoDto requestRegister(String loginId, String loginPassword){
        // 해당 아이디와 비밀번호에 맞는 회원이 있는지 검증
        Member member = verifyMember(loginId, loginPassword);   // ← 비번 검증 추가

        // 이미 패스워드리스에 등록되어있으면 막기
        if (client.isRegistered(loginId)) {
            throw new IllegalStateException("이미 패스워드리스에 등록된 계정입니다.");
        }

        // 패스워드리스에 등록되어있지 않으면 등록 진행
        PwlessResponse res = client.joinAp(loginId, member.getName(), member.getEmail());
        if (res == null || !res.isResult() || res.getData() == null) {
            throw new IllegalStateException("등록 정보 요청 실패");
        }
        JsonNode d = res.getData();
        // qr 화면 생성 및 운용하기 위해 프론트에서 필요한 데이터들
        return new RegisterInfoDto( // JsonNode 타입이여서 자바에서 사용하려면 변환을 해야줘야함 (String으로 변환)
                d.path("qr").asText(),
                d.path("serverUrl").asText(),
                d.path("registerKey").asText()
        );
    }

    // ===== 등록 완료 확인: 모바일 QR 등록 후 isAp로 확인 + DB 플래그 ON =====
    // (이 단계는 직전에 requestRegister로 본인 확인이 끝난 흐름의 연속이라 비번 재검증은 생략.
    //  isAp가 true여야만 플래그를 켜므로, 미등록 상태에선 아무 변화도 없다.)
    @Transactional
    public boolean confirmRegister(String loginId){
        boolean registered = client.isRegistered(loginId);
        if (registered) {
            Member member = memberRepository.findByLoginId(loginId)
                    .orElseThrow(() -> new IllegalArgumentException("존재하지 않는 아이디입니다."));
            // member 엔티티에 만들어놓은 패스워드리스 등록여부를 바꿈
            member.setPasswordlessEnabled(true);
        }
        return registered;
    }

    // ===== 로그인 1단계: 인증 시작(자동패스워드 생성) =====
    // 사용자가 ID를 넣고 패스워드리스로 로그인을 누르면 호출됨
    // 화면에 띄울 숫자(ex) 173 496)를 만들어오는 단계
    @Transactional(readOnly = true)
    public AuthStartDto startLogin(String loginId, String clientIp){
        // 등록된 회원이 아니면 예외 처리
        if (!client.isRegistered(loginId)) {
            throw new IllegalStateException("패스워드리스 미등록 계정입니다.");
        }

        // 등록된 회원이면 계속진행
        // 패스워드리스 서버에 보낼 매개변수들 생성하기
        // 1. 복호화된 일회성 토큰 가져오기
        String token = client.getDecryptedToken(loginId);
        // 2. 인증 결과 검증용 임의값 (서버로 보내는 내부 값, 화면에 안 보임)(servicePassword의 식별표 기능도 함)
        String random = UUID.randomUUID().toString().replace("-", "");
        // 3. 해당 로그인 시도가 승인해 줄 로그인 시도가 맞는지 식별해줄 세션아이디
        String sessionId = System.currentTimeMillis() + "-" + random.substring(0, 8);
        PwlessResponse res = client.getSp(token, loginId, random, sessionId, clientIp);
        if (res == null || !res.isResult() || res.getData() == null) {
            throw new IllegalStateException("인증 시작 실패: "
                    + (res != null ? res.getMsg() : "no response"));
        }
        JsonNode d = res.getData();
        return new AuthStartDto(
                sessionId,
                d.path("servicePassword").asText(), // 화면에 보여줄 자동패스워드 숫자 (서버가 생성)
                d.path("term").asInt(60)
        );
    }

    // ===== 로그인 2단계: 모바일 승인 결과 폴링 =====
    // + JWT 발급
    // 프론트가 2초마다 반복 호출하는 폴링 대상.
    // "승인됐어?" 확인하고, 승인됐으면 로그인 토큰을 발급하는 단계. 가장 중요한 메서드.
    // 승인여부는 data안에 있는 auth키의 값으로 결정함 (W, N, Y)
    @Transactional(readOnly = true)
    public AuthResultDto checkLogin(String loginId, String sessionId){
        PwlessResponse res = client.result(loginId, sessionId);
        if (res == null || res.getData() == null) {
            return new AuthResultDto("W", null);   // 아직 대기로 간주
        }

        // asText()안에 있는 "W"는 기본값임 (기본값 설정임) (auth가 없으면 "W"를 기본값으로 해라)
        String auth = res.getData().path("auth").asText("W");

        if ("Y".equals(auth)) { // 승인을 받았으면
            // 해당 loginId에 해당하는 Member의 정보를 가져와서 객체에 넣고
            Member member = memberRepository.findByLoginId(loginId)
                    .orElseThrow(() -> new IllegalArgumentException("존재하지 않는 아이디입니다."));
            // 그 member에게 token을 생성해줌
            String accessToken = jwtUtil.createAccessToken(
                    member.getMemberId(),
                    member.getLoginId(),
                    member.getName(),
                    member.getRole().name()
            );
            // AuthResultDto에 맞게 status랑 accessToken 값을 넣어줌
            return new AuthResultDto("Y", accessToken);
        }

        // 승인을 받지 않았으면
        return new AuthResultDto(auth, null);   // N(거절) 또는 W(대기)
    }



    // ===== 로그인 취소 =====
    public void cancelLogin(String loginId, String sessionId){
        client.cancel(loginId, sessionId);
    }

    // ===== 해지 =====
    // 패스워드리스 등록 해지
    // password를 추가로 받아 본인 확인 (공식 데모: 해지 시 암호 입력 요구)
    @Transactional
    public void withdrawal(String loginId, String loginPassword){
        // 해당 아이디와 비밀번호의 member 객체는 필요없어서 넣지 않음
        verifyMember(loginId, loginPassword);   // ← 비번 검증 추가

        boolean ok = client.withdrawal(loginId);
        if (ok) { // 해지되었으면
            // ifPresent : 값이 있으면 실행해라
            memberRepository.findByLoginId(loginId).ifPresent(m ->
                    m.setPasswordlessEnabled(false));
        }
    }

    // 단순 등록 여부 조회용 함수
    @Transactional(readOnly = true)   // 읽기 전용 — DB 변경 안 함
    public boolean isRegistered(String loginId){
        return client.isRegistered(loginId);   // 그냥 물어보고 결과만 반환
    }
}
```

```java
        return ResponseEntity.ok(Map.of("registered", passwordlessService.isRegistered(loginId)));
    }
```

```java
package com.backend_semi.passwordless.config;

import com.fasterxml.jackson.databind.ObjectMapper;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.http.MediaType;
import org.springframework.http.codec.json.Jackson2JsonDecoder;
import org.springframework.http.codec.json.Jackson2JsonEncoder;
import org.springframework.web.reactive.function.client.WebClient;

@Configuration
public class PasswordlessClientConfig {

    // 인증서버 호출 전용 WebClient
    @Bean
    // application.properties파일에 있는 데이터를 변수로 만들어서 넣은 PasswordlessProperties.java에 있는
    // 변수들을 매개변수로 사용하는데
    // 거기에 있는 패스워드리스 인증 서버인 authServerUrl를 baseUrl로 사용해서
    // 나중에 Controller에서 매핑을 할때 사용하는 주소에 하나하나 auth.jesuisben.store:11040를
    // 넣을 필요없이 바로 그 뒤의 주소를 넣을 수 있게 사용함
    // 프론트의 axiosInstance에 axios.create같은 것
    // 나중에 이 도구를 패스워드리스를 호출할때 사용함
    public WebClient passwordlessWebClient(PasswordlessProperties props){
        // WebClient가 text/plain 응답을 PwlessResponse로 변환 못해서
        // new Jackson2JsonDecoder(mapper, MediaType.APPLICATION_JSON, MediaType.TEXT_PLAIN)
        // 위의 코드를 넣어서 application/json이랑 text/plain 둘 다 JSON으로 읽으라고 설정하는 것
        ObjectMapper mapper = new ObjectMapper();

        return WebClient.builder()
                .baseUrl(props.getAuthServerUrl()) // http://auth.jesuisben.store:11040
                .codecs(config -> {
                    // JSON 디코더가 application/json 뿐 아니라 text/plain 응답도
                    // JSON으로 읽도록 허용 (이 인증서버는 JSON을 text/plain으로 보냄)
                    config.defaultCodecs().jackson2JsonDecoder(
                            new Jackson2JsonDecoder(mapper,
                                    MediaType.APPLICATION_JSON,
                                    MediaType.TEXT_PLAIN));
                    config.defaultCodecs().jackson2JsonEncoder(
                            new Jackson2JsonEncoder(mapper));
                })
                .build();
    }
}
```

```typescript
    const [registerKey, setRegisterKey] = useState(""); // 등록 코드(표시용)

    const [settingMsg, setSettingMsg] = useState("");
    const confirmRef = useRef<number | null>(null);                // 등록확인 폴링 타이머

    // 컴포넌트가 사라질 때 돌고 있는 타이머들 정리 (메모리 누수/유령 폴링 방지)
    useEffect(() => {
        return () => {
            if (pollRef.current) window.clearInterval(pollRef.current);
            if (confirmRef.current) window.clearInterval(confirmRef.current);
            if (termRef.current) window.clearInterval(termRef.current); // term 타이머 초기화
        };
    }, []);

    // 기존 로그인 (아이디, 비밀번호)
    const handlePasswordLogin = async (event?: React.SyntheticEvent) => {
        event?.preventDefault(); // 새로고침 방지
        console.log('로그인 시도중입니다.');

        try {
            const url = "/api/members/login";
            const params = { loginId, password }; // 파라미터
            const config = {
                headers: { // 헤더에 MIME type 적어서 요청
                    "Content-Type": "application/json"
                }
            };

            const response = await customAxios.post<LoginResponse>(url, params, config);

            console.log('응답 데이터 : \n' + response.data);

            // 서버의 응답을 전개 연산자로 처리합니다.
            // accessToken는 JWT, userData는 User.ts으로 구성된 객체
            // accessToken은 변수로 가져오고 ...userData는 객체로 가져옴
            const { accessToken, ...userData } = response.data;

            // localStorage에는 문자열만 들어갈 수 있음
            // 전개 연산자로 변수로 가져온 accessToken은 바로 넣을 수 있음
            localStorage.setItem("accessToken", accessToken);

            console.log('로그인 성공 사용자 : ' + userData);

            // 함수를 조건식에 넣는 것은 존재 유무를 판별하려고
            // (프롭스로 진짜 받아온 함수인가정도를 판단함)
            if (handleLoginSuccess) {
                handleLoginSuccess(userData);

                // userData는 자바스크립트 객체여서 문자열로 바꿔줘야 함
                // JSON.stringify 함수는 JavaScript 객체를 JSON 문자열로 변환해 줍니다.
                // App.tsx에서 로컬스토리지에 저장할 예정이라 여기서는 안해도 됨
                // 이제 로컬용이 아니고 배포도 해야해서 로컬스토리지에서 user 정보 보여주지 않음
                // localStorage.setItem("user", JSON.stringify(userData));
            }

            // 로그인이 되면 메인 홈페이지로 이동시킴
            navigate("/");

        } catch (error: any) {
            if (error.response) { // 서버가 에러 응답을 보냈을때
                // 백엔드에서 작성한 에러 메시지
                setErrors(error.response.data.message || "로그인 실패");
            } else { // 서버가 에러 응답을 안보냈을때 - 네트워크 문제
                setErrors("Server Error");
            }
        }
    };

    // ===================== 패스워드리스 로그인 =====================
    // 1) 인증 시작 → 숫자 표시 → 2초마다 결과 폴링
    const handlePwlessLogin = async () => {
        if (!loginId) {
            setPwlessMsg("아이디를 먼저 입력하세요.");
            return;
        }
        setPwlessMsg("");
        try {
            // 응답 데이터 : sessionId, servicePassword, term
            const res = await loginStartPwless(loginId);
            const { sessionId, servicePassword, term } = res.data;
            sessionRef.current = sessionId;
            setPwlessNum(servicePassword);                 // 화면에 173 496 표시
            setPwlessTerm(term); // 타이머용 state 변수
            setPwlessMsg("모바일 앱에서 이 번호를 확인하고 승인하세요.");

            // term 타이머 : 1초마다 남은 시간 1씩 감소
            termRef.current = window.setInterval(() => {
                setPwlessTerm((prev) => {
                    if (prev === null || prev <= 1) {
                        if (termRef.current) {
                            window.clearInterval(termRef.current);
                            termRef.current = null;
                        }
                        return 0;
                    }
                    return prev - 1;
                });
            }, 1000);

            // 2초마다 결과 폴링 시작
            pollRef.current = window.setInterval(pollOnce, 2000);
        } catch (error: any) {
            // 미등록 계정 등 → 서버가 보낸 메시지 표시
            setPwlessMsg(error.response?.data?.message || "패스워드리스 로그인 시작 실패");
        }
    };

    // 폴링 1회: 승인(Y)이면 로그인 완료, 거절(N)이면 중단, 대기(W)면 계속
    const pollOnce = async () => {
        try {
            // 응답 데이터 : status(W, N, Y), accessToken
            const res = await loginResultPwless(loginId, sessionRef.current);
            const { status, accessToken, ...userData } = res.data;

            if (status === "Y") { // 승인 되었으면
                stopPwlessLogin(); // 폴링 정리 (지속적 요청 중단)
                localStorage.setItem("accessToken", accessToken);
                // 토큰만으로 유저 정보를 복원하려면 별도 처리가 필요하지만,
                // 기존 흐름과 동일하게 토큰 저장 후 홈으로 이동 (App이 토큰 보고 상태 갱신)

                // 함수를 조건식에 넣는 것은 존재 유무를 판별하려고
                // (프롭스로 진짜 받아온 함수인가정도를 판단함)
                if (handleLoginSuccess) {
                    handleLoginSuccess(userData);
                }

                navigate("/");

            } else if (status === "N") {
                stopPwlessLogin();
                setPwlessMsg("인증이 거절되었습니다.");
            }
            // status === "W" 면 아무것도 안 함 (계속 폴링)
        } catch {
            // 일시 오류는 무시하고 다음 폴링 대기 (네트워크 깜빡임 대비)
        }
    };

    // 폴링 중단 + 화면 초기화
    const stopPwlessLogin = () => {
        if (pollRef.current) {
            window.clearInterval(pollRef.current);
            pollRef.current = null;
        }
        if (termRef.current) { // 타이머용 초기화
            window.clearInterval(termRef.current);
            termRef.current = null;
        }
    };

    // 사용자가 "취소" 누름 → 서버에 취소 알리고 폴링 정리
    const handlePwlessCancel = async () => {
        stopPwlessLogin();
        if (sessionRef.current) {
            try { await loginCancelPwless(loginId, sessionRef.current); } catch { }
        }
        setPwlessNum(null);
        setPwlessTerm(null); // 타이머용 취소 초기화
        setPwlessMsg("");
        sessionRef.current = "";
    };

    // 로그인 버튼: 모드에 따라 분기
    const handleLoginClick = () => {
        if (mode === "password") handlePasswordLogin();
        else handlePwlessLogin();
    };

    // ===================== "Passwordless설정" 모달: 등록 =====================
    // ID/PW 입력 → 등록(QR) 요청 → QR 표시 → 등록완료 폴링(앱 스캔 대기)
    const handleRegister = async () => {
        if (!setId || !setPw) {
            setSettingMsg("아이디와 비밀번호를 입력하세요.");
            return;
        }
        setSettingMsg("");
        try {
            // qr, serverUrl, registerKey 응답 받음
            const res = await registerPwless(setId, setPw);
            const { qrDataUrl, serverUrl, registerKey } = res.data;
            setQrDataUrl(qrDataUrl);
            setServerUrl(serverUrl);
            setRegisterKey(registerKey);
            setSettingMsg("앱으로 QR을 스캔한 뒤 등록을 완료하세요.");

            // 등록 완료(모바일 스캔) 폴링 — 2초마다 confirm 호출
            confirmRef.current = window.setInterval(async () => {
                try {
                    const c = await registerConfirmPwless(setId);
                    if (c.data.registered) {
                        stopConfirm();
                        setSettingMsg("패스워드리스 등록이 완료되었습니다!");
                        setQrDataUrl(null);
                    }
                } catch { }
            }, 2000);
        } catch (error: any) {
            setSettingMsg(error.response?.data?.message || "등록 요청 실패");
        }
    };

    // ===================== "Passwordless설정" 모달: 해지 =====================
    const handleWithdrawal = async () => {
        if (!setId || !setPw) {
            setSettingMsg("아이디와 비밀번호를 입력하세요.");
            return;
        }
        setSettingMsg("");
        try {
            await withdrawalPwless(setId, setPw);
            setSettingMsg("패스워드리스 등록이 해지되었습니다.");
        } catch (error: any) {
            setSettingMsg(error.response?.data?.message || "해지 실패");
        }
    };

    const stopConfirm = () => {
        if (confirmRef.current) {
            window.clearInterval(confirmRef.current);
            confirmRef.current = null;
        }
    };

    // 모달 닫기 + 상태 초기화
    const closeSetting = () => {
        stopConfirm();
        setSettingMode(null);
        setSetId("");
        setSetPw("");
        setQrDataUrl(null);
        setServerUrl(null);
        setRegisterKey("");
        setSettingMsg("");
    };

    // ============================== 화면 ==============================
    return (
        <div className="login-page">
            <div className="login-box">
                <h1 className="login-title">풀스택 강의실</h1>
                <p className="login-subtitle">로그인 후 강의를 이용할 수 있습니다.</p>

                <div className="login-form">
                    {errors && <Alert variant="danger">{errors}</Alert>}

                    <label className="login-label">아이디</label>
                    <input
                        className="login-input"
                        type="text"
                        placeholder="아이디를 입력하세요."
                        value={loginId}
                        onChange={(e) => setLoginId(e.target.value)}
                    />

                    {/* 비밀번호: 패스워드리스 모드에선 숨김 (공식 데모처럼 PW칸 비활성) */}
                    {mode === "password" && (
                        <>
                            <label className="login-label">비밀번호</label>
                            <input
                                className="login-input"
                                type="password"
                                placeholder="비밀번호를 입력하세요."
                                value={password}
                                onChange={(e) => setPassword(e.target.value)}
                                onKeyDown={(e) => { if (e.key === "Enter") handlePasswordLogin(); }}
                            />
                        </>
                    )}

                    {/* 로그인 방식 라디오 (공식 데모: Password / Passwordless) */}
                    <div className="login-mode-radios">
                        <label>
                            <input
                                type="radio"
                                name="loginMode"
                                checked={mode === "password"}
                                onChange={() => setMode("password")}
                            />{" "}
                            Password
                        </label>
                        <label style={{ marginLeft: 16 }}>
                            <input
                                type="radio"
                                name="loginMode"
                                checked={mode === "passwordless"}
                                onChange={() => setMode("passwordless")}
                            />{" "}
                            Passwordless
                        </label>
                    </div>

                    {/* 패스워드리스 로그인 진행 중: 자동패스워드 + 취소 */}
                    {pwlessNum && (
                        <div className="pwless-num-box">
                            <div className="pwless-num">{pwlessNum}</div>
                            {pwlessTerm !== null && ( // term 타이머용
                                <p className="pwless-msg">남은 시간: {pwlessTerm}초</p>
                            )}
                            <button className="login-button" onClick={handlePwlessCancel}>
                                취소
                            </button>
                        </div>
                    )}
                    {pwlessMsg && <p className="pwless-msg">{pwlessMsg}</p>}

                    {/* 로그인 버튼 (폴링 중이 아닐 때만) */}
                    {!pwlessNum && (
                        <button className="login-button" onClick={handleLoginClick}>
                            로그인
                        </button>
                    )}

                    {/* 하단 링크: 회원가입 | Passwordless설정 */}
                    <p className="login-signup-text">
                        <span
                            className="login-signup-link"
                            onClick={() => navigate("/members/signup")}
                        >
                            회원가입
                        </span>
                        {"  |  "}
                        <span
                            className="login-signup-link"
                            onClick={() => setSettingMode("register")}
                        >
                            Passwordless설정
                        </span>
                    </p>
                </div>
            </div>

            {/* ===================== Passwordless설정 모달 ===================== */}
            {
                settingMode && (
                    <div className="pwless-modal-backdrop" onClick={closeSetting}>
                        <div className="pwless-modal" onClick={(e) => e.stopPropagation()}>
                            <div className="pwless-modal-header">
                                <h2>Passwordless 설정</h2>
                                <button className="pwless-modal-close" onClick={closeSetting}>×</button>
                            </div>

                            {/* 등록 / 해지 탭 */}
                            <div className="pwless-tab">
                                <button
                                    className={settingMode === "register" ? "active" : ""}
                                    onClick={() => { setSettingMode("register"); setSettingMsg(""); }}
                                >
                                    등록
                                </button>
                                <button
                                    className={settingMode === "withdrawal" ? "active" : ""}
                                    onClick={() => { setSettingMode("withdrawal"); setSettingMsg(""); setQrDataUrl(null); }}
                                >
                                    해지
                                </button>
                            </div>

                            {/* QR이 떠 있지 않을 때만 입력칸 표시 */}
                            {!qrDataUrl && (
                                <>
                                    <input
                                        className="login-input"
                                        type="text"
                                        placeholder="아이디"
                                        value={setId}
                                        onChange={(e) => setSetId(e.target.value)}
                                    />
                                    <input
                                        className="login-input"
                                        type="password"
                                        placeholder="비밀번호"
                                        value={setPw}
                                        onChange={(e) => setSetPw(e.target.value)}
                                    />
                                    {settingMode === "register" ? (
                                        <button className="login-button" onClick={handleRegister}>
                                            등록 (QR 받기)
                                        </button>
                                    ) : (
                                        <button className="login-button" onClick={handleWithdrawal}>
                                            해지하기
                                        </button>
                                    )}
                                </>
                            )}

                            {/* 등록 QR 표시 */}
                            {qrDataUrl && (
                                <div className="pwless-qr-box">
                                    <p>스마트폰 앱에서 아래 QR을 스캔하세요.</p>
                                    <img src={qrDataUrl} alt="등록 QR" className="pwless-qr-img" />
                                    {serverUrl && <p>서버 URL: {serverUrl}</p>}
                                    {registerKey && <p>등록 코드: {registerKey}</p>}
                                </div>
                            )}

                            {settingMsg && <p className="pwless-msg">{settingMsg}</p>}
                        </div>
                    </div>
                )
            }
        </div >
    );
}

export default LoginPage;
```

```yml
name: Spring CD - Deploy to EC2
# CI 워크플로우("Spring CI - Build and Push")가 성공적으로 끝나면 자동 실행
on:
  workflow_run:
    workflows: ["Spring CI - Build and Push"]
    types:
      - completed
jobs:
  deploy:
    runs-on: ubuntu-latest
    # CI가 성공(success)했을 때만 배포 진행
    if: ${{ github.event.workflow_run.conclusion == 'success' }}
    steps:
      # EC2에 SSH 접속해서 배포 명령 실행
      - name: Deploy to EC2
        uses: appleboy/ssh-action@v1.0.3
        with:
          host: ${{ secrets.EC2_HOST }}
          username: ubuntu
          key: ${{ secrets.EC2_KEY }}
          script: |
            # 1. Docker Hub 로그인
            echo "${{ secrets.DOCKER_PASSWORD }}" | sudo docker login -u "${{ secrets.DOCKER_USERNAME }}" --password-stdin
            # 2. 최신 이미지 받기
            sudo docker pull "${{ secrets.BACKEND_IMAGE }}"
            # 3. 기존 백엔드 컨테이너 삭제 (없으면 그냥 넘어감)
            sudo docker rm -f backend-app || true
            # 4. 새 백엔드 컨테이너 실행
            #    --network host: EC2 네트워크 공유 → localhost:3306으로 mysql-db에 연결
            sudo docker run -d \
              --name backend-app \
              --network host \
              --restart unless-stopped \
              -e DB_URL="${{ secrets.DB_URL }}" \
              -e DB_USERNAME="${{ secrets.DB_USERNAME }}" \
              -e DB_PASSWORD="${{ secrets.DB_PASSWORD }}" \
              -e JWT_PRIVATE_KEY="${{ secrets.JWT_PRIVATE_KEY }}" \
              -e JWT_PUBLIC_KEY="${{ secrets.JWT_PUBLIC_KEY }}" \
              -e S3_BUCKET="${{ secrets.S3_BUCKET }}" \
              -e S3_REGION="${{ secrets.S3_REGION }}" \
              -e S3_ACCESS_KEY="${{ secrets.S3_ACCESS_KEY }}" \
              -e S3_SECRET_KEY="${{ secrets.S3_SECRET_KEY }}" \
              -e PWLESS_SERVER_KEY="${{ secrets.PWLESS_SERVER_KEY }}" \
              "${{ secrets.BACKEND_IMAGE }}"
            # 5. 안 쓰는 옛날 이미지 정리
            sudo docker image prune -f
```

#### \# 롬복(Lombok) 사용
롬복 구현 절차

step01. Lombok 플러그인 설치
Lombok 플러그인은 IDE(인텔리 제이)가 Lombok이 생성한 메소드/생성자를 인식할 수 있게 해주는 역할을 합니다.
설정 메뉴에서 Plugins 영역의 Marketplace에서 'lombok'로 검색을 하고, Install 버튼을 클릭합니다.

step02. annotation processing 설정
자바에서는 @Override, @Deprecated 같은 기본 어노테이션뿐 아니라, 개발자가 정의한 어노테이션을 사용할 수 있습니다.
그런데 어떤 어노테이션은 단순히 표시 역할만 하는 게 아니라, 컴파일 시점에 코드를 새로 생성하거나 수정하는 기능을 합니다.
이를 담당하는 것이 `어노테이션 프로세서(annotation processor)`입니다.

설정 메뉴의 Build, Execution, Deployment → Compiler → Annotation Processors → Enable annotation processing
을 켜 주면, IDE가 컴파일 시 Lombok 같은 어노테이션 프로세서를 실행하도록 허용하는 겁니다.

#### \# 군집 분석 (머신 러닝 교안(이론).pdf (P.259~262))
데이터 상호간의 유사성(similarity)을 토대로 그 값이 가까운 것부터 순서대로 합쳐 군집을 만들어 가는 기법
(유유상종)
대표적인 비지도 학습 (input 데이터만 넣고 정답(라벨)은 넣지 않는 학습 방법)

- 특징
전체적인 데이터 구조를 파악하는 데 이용
유사한 특성을 가진 항목들을 합쳐 가면서 유사한 특성을 군집을 찾아냄
분류하려고 하는 항목에 대한 사전 지식이 없는 상태에서 이루어지므로 비지도 학습
유사도에 근거하여 군집들을 분석하는 데, 유사성은 Euclidean(유클리디언) 거리 사용
유클리디언 거리 : 쉽게 말하면 두 점 사이의 직선 거리 (2차원에서는 피타고라스 정리 사용)
분석 결과에 대한 가설 검정이 없음
규칙을 기반으로 계층적인 트리 구조를 생성

참고로 코사인 유사도도 사용함

- 유클리디언 거리 vs 코사인 유사도
1) 유클리디언 거리
	"얼마나 가까이 있는가?"를 측정
	직선 거리를 계산

2) 코사인 유사도
	"방향이 얼마나 같은가?"를 측정
	벡터가 이루는 각도를 이용
		1 : 방향이 완전히 같다.
		0 : 서로 관계가 없다(직각).
		-1 : 정반대 방향이다.


- 클러스터링 방법 (머신 러닝 교안(이론).pdf (P.262))
클러스터링 : 비슷한 데이터를 군집(Cluster)으로 묶는 것
클러스터링 방법 : 어떻게 묶을 것인가?

| 항목 | 설명 | 키워드 |
|------|------|---------|
| 최단 연결법 | 군집 A와 군집 B에 속하는 데이터 중에서 가장 가까운 데이터들을 군집으로 묶는 방법입니다. | single |
| 최장 연결법 | 군집 A와 군집 B에 속하는 데이터 중에서 가장 먼 데이터들을 군집으로 묶는 방법입니다. 완전 연결법이라고도 합니다. | complete |
| 워드 연결법 | 군집을 묶을 때 생기는 새로운 군집의 오차 제곱의 합을 이용하는 방식입니다. | ward.D2 |
| 평균 연결법 | 군집 A의 요소와 군집 B의 요소 각각의 거리를 모두 구하여, 이것의 평균값을 이용하는 방식입니다. | average |

- 코드 내용
```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans

# 데이터
X = np.array([
    [3, 3],
    [4, 4],
    [3, 5],
    [5, 4],
    [4, 2],
    [7, 7],
    [8, 8],
    [7, 9],
    [6, 8],
    [8, 7]
])

# 초기 중심점
initial_centers = np.array([
    [4, 5],
    [6, 6]
])

# K-Means 수행
kmeans = KMeans(
    n_clusters=2,
    init=initial_centers,
    n_init=1,
    random_state=42
)

kmeans.fit(X)

# 결과 출력
print("=== 최초 중심점 ===")
for i, center in enumerate(initial_centers):
    print(f"중심점{i}: {center}")

print("\n=== 갱신된 중심점 ===")
for i, center in enumerate(kmeans.cluster_centers_):
    print(f"중심점{i}: {center}")

# --------------------
# 시각화
# --------------------
plt.figure(figsize=(8, 6))

colors = ['royalblue', 'orange']

# 군집별 데이터 표시
for cluster in range(2):
    cluster_points = X[kmeans.labels_ == cluster]

    plt.scatter(
        cluster_points[:, 0],
        cluster_points[:, 1],
        s=120,
        color=colors[cluster],
        label=f'Cluster {cluster}'
    )

# 데이터 번호 표시
for i, point in enumerate(X, start=1):
    plt.text(
        point[0] + 0.1,
        point[1] + 0.1,
        str(i),
        fontsize=10
    )

# 초기 중심점
plt.scatter(
    initial_centers[:, 0],
    initial_centers[:, 1],
    marker='X',
    s=350,
    color='red',
    label='Initial Centers'
)

# 갱신된 중심점
plt.scatter(
    kmeans.cluster_centers_[:, 0],
    kmeans.cluster_centers_[:, 1],
    marker='X',
    s=350,
    color='black',
    label='Updated Centers'
)

# 중심점 이동 경로
for old, new in zip(initial_centers, kmeans.cluster_centers_):
    plt.arrow(
        old[0],
        old[1],
        new[0] - old[0],
        new[1] - old[1],
        length_includes_head=True,
        head_width=0.15,
        linestyle='--'
    )

plt.title("K-Means Clustering (K=2)", size=20)
plt.xlabel("x1")
plt.xlim([0, 10])
plt.ylim([0, 10])
plt.xticks(np.arange(0, 11, 1))
plt.yticks(np.arange(0, 11, 1))
plt.ylabel("x2")
plt.grid(True)
plt.legend()
# plt.show()
plt.savefig("../dataOut/" + "kmeans_example.png")
```



#### \# pdist() 함수 (머신 러닝 교안(이론).pdf (P.126))
주어진 관측치 집합의 각 쌍 간 거리를 계산해 주는 함수
일반적으로, 두 데이터 포인트 간의 거리를 효율적으로 계산하기 위해 사용이 되며, 주로 군집화(clustering) 분석이나 패턴 인식에서 많이 사용
관측치 간 유클리드 거리, 코사인 유사도 등을 계산할 때 매우 유용

| 항목 | 설명 |
|------|------|
| 사용 예시 | `distances = pdist(x, metric='euclidean')` |
| x | 관측치의 2차원 배열. |
| metric | 거리 계산에 사용할 메트릭을 지정합니다.<br>기본값은 `'euclidean'`이며, 다양한 메트릭(예: `'minkowski'`, `'cityblock'`, `'cosine'`, `'hamming'`)을 지원합니다. |
| distances | 반환 값은 각각의 관측치 쌍 사이의 거리를 1차원 배열로 반환해 줍니다. |



#### \# 다차원 척도법(MDS) (머신 러닝 교안(이론).pdf (P. 273))
고차원 공간의 데이터 간의 거리를 유지하면서, 저차원(2/3차원 공간)에 표현하여 시각화할 수 있도록 도와주는 기법

- 특징
	고차원 공간의 데이터 포인트들 간의 거리(또는 유사성)를 유지하면서,
	이를 2차원 또는 3차원 공간에 표현하여 시각화할 수 있도록 도와 주는 기법입니다.
	주로 데이터 포인트들 간의 관계를 시각적으로 이해하기 위해 사용됩니다.
	예를 들어, 문서들 간의 유사성, 제품 간의 거리 등을 시각화할 수 있습니다.

- 매개변수

| 항목            | 설명                                                                                                                                                                                      |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 사용 형식         | `mds = MDS(n_components=2, dissimilarity='precomputed', random_state=42)`                                                                                                               |
| n_components  | 임베딩할 목표 차원 수입니다. 보통 2차원 또는 3차원으로 설정합니다.<br>예시) `n_components=2`는 2차원 공간에 임베딩합니다.                                                                                                        |
| dissimilarity | 거리(또는 유사성) 측정 방법입니다.<br>기본값은 `'euclidean'`이며, `'precomputed'`로 설정하면 사전에 계산된 거리 행렬을 사용할 수 있습니다.<br>예시) `dissimilarity='precomputed'`는 이미 계산된 거리 행렬을 사용하여 MDS를 수행합니다.                     |
| metric        | MDS의 종류를 결정하는 매개변수입니다.<br>`True`로 설정하면 유클리드 거리 기반의 metric MDS를 수행하고, `False`로 설정하면 non-metric MDS를 수행합니다.<br><br>metric MDS: 거리 자체를 최대한 유지하는 방식<br>non-metric MDS: 거리의 순서(순위)만을 유지하는 방식 |
| random_state  | 결과의 재현성을 위해 난수 생성기를 고정합니다.                                                                                                                                                              |
| 사용 예시         | # 다차원 척도법(MDS)<br>`mds_fit = mds.fit_transform(squareform(distance))`                                                                                                                   |

- 코드 내용
```python
from scipy.cluster.hierarchy import linkage, dendrogram
from scipy.spatial.distance import pdist, squareform
from sklearn.manifold import MDS

data = [[2, 5, 3, 4, 4], [2, 4, 3, 3, 5], [4, 5, 3, 4, 4], [5, 6, 3, 3, 3], [4, 1, 3, 2, 2]]

import numpy as np
matrix = np.array(data)

import pandas as pd

mycolumns = ['달달함', '목넘김', '고소함', '기름짐', '매콤함']
myindex = ['짜장면', '짬뽕', '라면', '우동', '돈가스']
df = pd.DataFrame(matrix, index=myindex, columns=mycolumns)

print('데이터 정보')
print(df)

# 유클리디언 거리 알고리즘을 사용하여 거리를 계산 후 1차원 배열로 반환해 줍니다.
distance = pdist(df, metric='euclidean') # 유클리디언 거리
print('거리 정보(1차원 배열)')
print(distance) # 나올 수 있는 경우의 수는 수학에서 '조합' 공부하기

# squareform : 1차원 거리 벡터를 대칭 행렬(2차원) 형태로 변환해 줍니다.
distance_matrix = pd.DataFrame(squareform(distance), index=df.index, columns=df.index)
print('거리 행렬')
print(distance_matrix)

new_distance = distance ** 2
distance_matrix = pd.DataFrame(squareform(new_distance), index=df.index, columns=df.index)
print('거리 행렬')
print(distance_matrix)

linked = linkage(distance, method='single') # 최단 거리 군집화

import matplotlib.pyplot as plt

dataIn, dataOut = './../dataIn/', './../dataOut/'

plt.rc('font', family='Malgun Gothic')
plt.rcParams['axes.unicode_minus'] = False

# 덴드로 그램 그리기
plt.figure(figsize=(8, 4))
plt.title('dendrogram(single linkage)')

dendrogram(linked, labels=df.index.tolist())
plt.savefig(dataOut + 'image01.png')

# 다차원 척도법
mds = MDS(n_components=2, dissimilarity='precomputed', random_state=42)

mds_fit = mds.fit_transform(squareform(distance))

plt.figure(figsize=(6, 6))
plt.scatter(mds_fit[:, 0], mds_fit[:, 1], color='red')
plt.plot([-2.5, 3], [0, 0], 'k--') # draw x-axia dotted line
plt.plot([0, 0], [-2.5, 3], 'k--') # draw y-axia dotted line

for idx, label in enumerate(df.index):
    plt.text(mds_fit[idx, 0], mds_fit[idx, 1], label, color='red')

plt.title('MDS Plot')

plt.savefig(dataOut + 'mds-plot.png')
```

- 개념
```text
CNN은 이미지 분류, 객체 감지, 얼굴 인식 등의 다양한 컴퓨터 비전 작업에 사용됩니다.
테두리, 선, 색 등 이미지의 시각적인 특징/특성을 좀 더 확실하게 감지하고자 하는 것이 목표입니다.
입력되는 데이터는 image로써 은닉 계층의 모든 뉴런은 가중치 행렬 w와 편향 b를 공유합니다.
```

- CNN(합성곱 신경망)
이미지 속 특징을 자동으로 찾아 물체를 인식하는 딥러닝 알고리즘
\* 이것은 이미지로 하는거지만, 텍스트로 딥러닝하는 것은 RNN이라고 함
컴퓨터는 이미지를 숫자의 집합(Pixel의 집합)으로 인식하고, 숫자들 속에서 특징(Feature)을 찾아냄

| 항목 | 설명 |
|------|------|
| 합성곱 계층 (Convolution Layer) | 입력 데이터(이미지)와 필터(커널)를 사용하여 특징 맵(feature map)을 생성합니다.<br>필터는 작은 윈도우로 입력 데이터를 슬라이딩하면서 특징을 감지하는 역할을 합니다.<br>컨볼루션 계층은 특징이 감지된 구조를 고려하여 입력 데이터의 특징을 추출합니다. |
| 풀링 계층 (Pooling Layer) | 풀링(Pooling)은 특징 맵의 크기를 줄이는 과정을 의미합니다.<br>합성곱 계층의 출력 맵을 단순히 압축(컴팩트)시켜서 계산 비용을 줄이기 위해 사용합니다.<br>특징 맵의 중요한 정보를 유지하면서 크기를 감소시킵니다.<br>주로 최대 풀링(max pooling)이나 평균 풀링(average pooling)을 사용합니다. |
| 완전 연결 계층 (Fully Connected Layer) | 컨볼루션 계층과 풀링 계층을 통해 얻은 특징을 기반으로 분류 등의 작업을 수행합니다.<br>완전 연결 계층은 이전 계층의 모든 뉴런과 연결되어 있는 계층으로, 일반적인 신경망의 형태를 가지고 있습니다. |
| Max Pooling | 해당 영역에서 가장 큰 요소를 찾아 내는 것을 말합니다. |
| kernel | 은닉 계층에서 공통적으로 사용하게 되는 가중치(w)와 편향(b)을 의미합니다.<br>filter라고도 부릅니다. |
| 특징 맵/특성 맵 | 하나의 커널이 이미지에서 한 종류의 특징을 감지해 내는 것을 의미합니다. |
| stride | kernel이 우측 또는 하단으로 한 번에 이동할 이동 개수입니다. |
| padding | 좀 더 좋은 결과를 내기 위하여 이미지의 외부에 데이터를 붙여주는 기능(숫자 0을 붙임)입니다. |

```python
from keras import Sequential  # Keras에서 순차적으로 레이어를 쌓는 모델을 가져옴
from keras.layers import Dense, Conv2D, MaxPooling2D, Flatten  # 필요한 레이어들 가져오기 (밀집 레이어, 2D 합성곱, 풀링, 평탄화)

# 입력되는 이미지
# channel : 흑백 이미지는 1, 컬러 이미지는 3
image_size = (150, 150, 3) # width, height, channel

# 머신 러닝 모델 객체 생성: Sequential()은 순차적으로 레이어를 쌓는 방법을 제공
model = Sequential()

# 첫 번째 Conv2D 레이어 (합성곱 레이어):
# 3행 3열짜리 필터 32개를 사용하여 입력 이미지 image_size에 대한 컨블루션 연산을 수행합니다.
# 활성화 함수는 relu 함수를 사용하도록 하겠습니다.
# - filters=32: 32개의 필터(특징 추출기)를 사용하여 입력 이미지를 분석
# - kernel_size=(3, 3): 3x3 크기의 필터(커널)를 사용
# - activation='relu': 활성화 함수로 ReLU(Rectified Linear Unit)를 사용하여 비선형성을 추가
# - input_shape=(150, 150, 3): 입력 이미지의 크기는 150x150 픽셀, 3채널(컬러 이미지)
model.add(Conv2D(filters=32, kernel_size=(3, 3), activation='relu', input_shape=image_size))

# MaxPooling 레이어:
# - pool_size=(2, 2): 2x2 크기의 필터를 사용하여 입력의 크기를 절반으로 줄이는 풀링(최대값 추출) 작업
model.add(MaxPooling2D(pool_size=(2, 2)))

# 두 번째 Conv2D 레이어:
# - filters=64: 64개의 필터를 사용하여 더 복잡한 특징을 추출
# - kernel_size=(3, 3): 3x3 크기의 필터
# - activation='relu': 활성화 함수로 ReLU 사용
model.add(Conv2D(filters=64, kernel_size=(3, 3), activation='relu'))

# 두 번째 MaxPooling 레이어:
# - pool_size=(2, 2): 2x2 크기의 필터를 사용하여 입력의 크기를 절반으로 줄임
model.add(MaxPooling2D(pool_size=(2, 2)))

# 세 번째 Conv2D 레이어:
# - filters=128: 128개의 필터를 사용하여 더 복잡한 특징을 추출
# - kernel_size=(3, 3): 3x3 크기의 필터
# - activation='relu': 활성화 함수로 ReLU 사용
model.add(Conv2D(filters=128, kernel_size=(3, 3), activation='relu'))

# 세 번째 MaxPooling 레이어:
# - pool_size=(2, 2): 2x2 크기의 필터로 입력의 크기를 절반으로 줄임
model.add(MaxPooling2D(pool_size=(2, 2)))

# 네 번째 Conv2D 레이어:
# - filters=128: 128개의 필터를 사용하여 더 복잡한 특징을 추출
# - kernel_size=(3, 3): 3x3 크기의 필터
# - activation='relu': 활성화 함수로 ReLU 사용
model.add(Conv2D(filters=128, kernel_size=(3, 3), activation='relu'))

# 네 번째 MaxPooling 레이어:
# - pool_size=(2, 2): 2x2 크기의 필터로 입력의 크기를 절반으로 줄임
model.add(MaxPooling2D(pool_size=(2, 2)))

# Flatten 레이어:
# 컨블루션 연산 결과를 1차원 형태의 배열로 만들어, 완전 연결(Dense) 레이어에 전달합니다.
model.add(Flatten())

# 완전 연결(Dense) 레이어:
# - units=512: 512개의 뉴런을 가진 밀집 레이어
# - activation='relu': 활성화 함수로 ReLU 사용
model.add(Dense(units=512, activation='relu'))


# 출력층(Output Layer)
# - units=1: 출력 뉴런의 수는 1 (이항 분류에서 사용할 것임)
# - activation='sigmoid': 활성화 함수로 sigmoid를 사용하여 출력값을 0과 1 사이로 제한 (이항 분류에 적합)
# 우리는 이항 분류(강아지/고양이)를 할 예정이므로, activation 함수는 반드시 sigmoid 함수를 사용해야 합니다.
# 주의) 다항 분류인 경우에는 반드시 one hot encoding을 수행해야 합니다.
y_column = 1
model.add(Dense(units=y_column, activation='sigmoid'))

# 모델의 구조를 출력해주는 함수로, 레이어의 상세 정보를 확인할 수 있음
#model.summary()
from Utility.keras_graph_util import model_information
model_information(model)
```

#### \# 케라스(Keras) - 드롭 아웃(DropOut) (머신 러닝 교안(이론).pdf (P. 147))
1) 개념
```text
인공 신경망에서 사용되는 정규화(regularization) 기법의 일종입니다.
학습 과정에서 촘촘한 Neural Net(신경망) 연결선의 일부를 무작위로 선택하여 없애 버리는 것을 말합니다.
제외된 유닛은 신경망의 계산 로직에 참여하지 않습니다.
dropout의 값으로 몇 %를 유지시킬 것인가에 대한 값을 지정해 주어야 합니다. 
```

2) 필요성
```text
네트워크의 복잡성을 줄여서 과적합(overfitting) 방지
모델의 일반화 성능 향상 기대
```

3) 주의 사항
```text
Dropout은 모델의 학습 단계에서만 사용되며, 테스트나 추론 과정에서는 비활성화됩니다.
실제 예측 시에는 Dropout을 적용하지 않고 전체 네트워크를 사용하여 예측을 수행해야 한다는 것입니다.
```

| 항목    | 설명                                                                                                     |
| ----- | ------------------------------------------------------------------------------------------------------ |
| 사용 문법 | model.add(Dropout(rate)) # rate는 드롭 아웃의 비율<br>각 학습 배치(batch)마다 rate%의 비율로 유닛이 무작위로 제외되어 학습에 사용되지 않습니다. |
| 주의 사항 | 학습(training) 시에는 값을 조절해서 사용합니다.<br>테스트 시에는 반드시 1의 값을 주어 모든 layer를 거쳐 테스트 할 수 있게 해야 합니다.                |
| 단점    | training 할 때, 시간이 더 많이 걸립니다.                                                                           |
| 기대 효과 | 보다 정확한 training이 가능하므로, 실제 테스트에서 보다 더 좋은 성능을 발휘합니다.                                                    |

```python
import sys
import pandas as pd

from keras import Sequential
from keras.datasets import mnist
from keras.utils import to_categorical
from keras.layers import Conv2D, Dense, MaxPool2D, Flatten, Dropout

menu = int(input("CNN은 숫자 6, CNN+DropOut은 숫자 7을 입력해 주세요 : "))

if menu < 6 or menu > 7:
    print("프로그램을 종료합니다.")
    sys.exit(0)

(x_train, y_train), (x_test, y_test) = mnist.load_data()

TARGET_WIDTH = x_train.shape[1]
TARGET_HEIGHT = x_train.shape[2]
COLOR_MODE = 1
NUMBER_CLASSES = 10

x_train.shape = x_train.shape + (1,)
x_train = x_train.astype(float) / 255.0

x_test.shape = x_test.shape + (1,)
x_test = x_test.astype(float) / 255.0

y_train = to_categorical(y_train, num_classes=NUMBER_CLASSES)
y_test = to_categorical(y_test, num_classes=NUMBER_CLASSES)

EPOCH_SU = 5
BATCH_SIZE = 64
VALIDATION_SPLIT = 0.3

model = Sequential()

KERNEL_SIZE = (3, 3)
POOL_SIZE = (2, 2)

model.add(Conv2D(filters=32, kernel_size=KERNEL_SIZE, activation="relu", input_shape=(TARGET_WIDTH, TARGET_HEIGHT,COLOR_MODE)))
model.add(MaxPool2D(pool_size=POOL_SIZE))

model.add(Conv2D(filters=64, kernel_size=KERNEL_SIZE, activation="relu"))
model.add(MaxPool2D(pool_size=POOL_SIZE))

model.add(Conv2D(filters=64, kernel_size=KERNEL_SIZE, activation="relu"))
model.add(MaxPool2D(pool_size=POOL_SIZE))

model.add(Flatten())

if menu == 6 :
    model.add(Dense(units=512, activation="relu"))
    model.add(Dense(units=512, activation="relu"))

elif menu == 7 :
    DROPOUT_RATE = 0.3

    model.add(Dense(units=512, activation="relu"))
    model.add(Dropout(DROPOUT_RATE))

    model.add(Dense(units=512, activation="relu"))
    model.add(Dropout(DROPOUT_RATE))
# end if

model.add(Dense(units=NUMBER_CLASSES, activation="softmax"))

model.compile(loss="categorical_crossentropy", optimizer="adam", metrics=["accuracy"])

fit_hist = model.fit(x_train, y_train, epochs=EPOCH_SU, batch_size=BATCH_SIZE, validation_split=VALIDATION_SPLIT, verbose=1)


print('history 객체가 소유하고 있는 키 목록 보기')
print(fit_hist.history.keys())

from Utility.keras_graph_util import model_information, graph_accuracy_validation, graph_loss_validation, graph_accuracy_loss

model_information(model)

print('모델의 성능 평가')
score = model.evaluate(x_test, y_test, verbose=0)
print(type(score))

# `손실 함수`와 `정확도`를 출력해 봅니다.
print(f'test loss : {score[0]}, test accuracy : {score[1]}')

# 차후를 위하여 별도의 파일로 기록해 둡니다.
dataOut = './../dataOut/'

filename_suffix = str(menu).zfill(2)

filename = dataOut + 'mnist_result_' + filename_suffix + '.csv'

df = pd.DataFrame(score).T
df.columns = ['loss', 'accuracy']
df.to_csv(filename, index=False)
print(filename + ' 파일이 저장되었습니다.')

# 유틸리티 모듈을 이용하여 그래프를 그립니다.
graph_accuracy_validation(fit_hist, 'mnist_accuracy_validation_' + filename_suffix)

graph_loss_validation(fit_hist, 'mnist_loss_validation_' + filename_suffix)

graph_accuracy_loss(fit_hist, 'mnist_accuracy_loss_' + filename_suffix)

print('finished')
```

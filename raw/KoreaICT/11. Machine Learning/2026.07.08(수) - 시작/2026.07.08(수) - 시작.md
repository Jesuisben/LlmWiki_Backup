#### \# 머신 러닝의 2가지 분류 (머신 러닝 교안(이론).pdf (P.10~23))
##### 1\. 학습 방법에 따른 분류
- 학습이란
데이터 기반, 특정 작업(분류, 회귀 등등)을 수행할 수 있도록 패턴이나 규칙을 스스로 찾아내는 과정을 의미

###### 1) 지도 학습
- 개념
정답을 알려주면서 공부시키는 방법
독립변수(학습자료)(Input)와 종속변수(정답)(Label)를 동시에 알려주고 학습시킴
ex) 
선형 회귀(Linear Regression)
로지스틱 회귀(Logistic Regression)
서포트 벡터 머신(Support Vector Machines, SVM)
의사 결정 나무(Decision Trees)
랜덤 포레스트(Random Forest)
신경망(Neural Networks)

- 목표
예측 문제
정답이 있는 데이터를 사용하여 학습하고, 새로운 데이터에 대해 예측하는 것


###### 2) 비지도 학습
- 개념
정답 없이 스스로 규칙을 찾게 하는 방법
독립변수(학습자료)(Input)를 알려주고, 종속변수(정답)(Label)는 알려주지 않고 학습시킴
ex) 
K-평균 군집화(K-Means Clustering)
계층적 군집화(Hierarchical Clustering)
주성분 분석(Principal Component Analysis, PCA)
독립 성분 분석(Independent Component Analysis, ICA)
자기 조직화 지도(Self-Organizing Maps, SOM)

- 목표
데이터 탐색과 분석
정답이 없는 데이터를 사용하여 데이터의 패턴이나 구조를 발견하는 것


###### 3) 강화 학습
- 개념
행동의 결과로 얻는 보상(Reward)을 바탕으로 스스로 최적의 행동을 학습하는 방법

- 목표
환경과의 상호 작용을 통한 최대화된 보상 받기


##### 2\. 문제의 유형에 따른 분류
- 데이터 분류

| 구분               | 정성/정량 관계              | 설명                  | 예시              |
| ---------------- | --------------------- | ------------------- | --------------- |
| 범주형(Categorical) | 정성적 데이터(qualitative)  | 수치보다는 구분(label)이 중요 | 성별, 지역, 직업, 혈액형 |
| 연속형(Continuous)  | 정량적 데이터(quantitative) | 수치로 연속적인 값 표현       | 키, 몸무게, 점수, 매출액 |

##### 3\. 분류 방식
###### 머신 러닝

#### \# 머신 러닝에 대한 용어들 (머신 러닝 교안(이론).pdf (P.28~))
- 추천(Recommendation)
사용자의 취향에 맞는 상품, 서비스, 콘텐츠 등을 예측하고 제공하는 기술

- 차원 축소
많은 데이터를 핵심 정보만 남겨 단순화 시키는 기법입니다.(고차원→저차원)

- 자연어 처리(Natural Language Processing, NLP)
인간이 사용하고 있는 언어/문자 등을 이용하는 기법
문서 분류, 감정 분석, 기계 번역, 질의 응답 시스템 등에 사용
ex) 스팸 메일 필터링, 챗봇, 문서 요약 등.

- 결측치

결측치가 있으면 분석 전에 반드시 탐지하고 적절히 처리하는 전처리 과정을 거쳐야 함

- 이상치(outlier)
데이터 집합에서 다른 값들의 데이터 패턴과 크게 벗어난 값을 의미
이러한 값들은 데이터의 일반적인 패턴과 동떨어져 있어 극단적인 값으로 간주
ex) 키가 대부분 150~180cm인데 한 명이 250cm라고 기록됨

- linear regression : 선형 회귀 (머신 러닝 교안(이론).pdf (P.153))
1) 단순 선형 회귀 : 하나의 독립 변수와 하나의 종속 변수로 가정
2) 다중 선형 회귀 : 여러 개의 독립 변수와 하나의 종속 변수를 가정

- 모델
주어진 훈련 데이터를 기반으로 회귀 계수(coefficient)를 추정하고, 이를 사용하여 새로운 입력 데이터에 대한 출력 값을 예측할 수 있음

- 회귀 분석 - R^2 결정 계수 (머신 러닝 교안(이론).pdf (P. 186))
	회귀 분석에서 모델의 설명력을 나타내는 통계 지표
	구하는 함수 : r_square = model.score(x_test, y_test)
	
1) R^2 = 1: 완벽한 모델
	독립 변수가 종속 변수의 변동을 완벽하게 설명하는 경우.
	(이상적인 상황) 독립 변수들이 종속 변수의 변동에 상당한 영향을 미치고 있다는 것을 의미.
2) R^2 = 0.5:
	회귀선이 데이터의 50% 변동성을 설명하고, 50%만 다른 요인(오차)로 설명됨.
3) R^2 = 0: 설득(설명)력이 전혀 없는 모델
	독립 변수가 종속 변수의 변동을 전혀 설명하지 못하는 경우.
	회귀선이 데이터 평균과 다를 바 없음.

# (공부할때 매번 값이 바뀌니까 불편. 진짜 할때는 시드값 없이 해야함)
x_train, x_test, y_train, y_test = \
    train_test_split(x, y, test_size=0.3, random_state=10)

print(f'학습용 데이터 셋 개수 : {len(x_train)}')
print(f'테스트용 데이터 셋 개수 : {len(x_test)}')

# 머신러닝의 실제 동작 순서 : 훈련 데이터 → fit() → 회귀 계수·절편 도출 → 회귀식 생성 → predict()로 예측
# 단순 회귀 분석 모델 객체 생성
# 모델 객체는 훈련 데이터를 학습(fit)하여
# 회귀 계수(기울기)와 절편을 도출하고,
# 이를 이용하여 새로운 입력 데이터의 값을 예측함.
# 현재는 객체만 생성한 상태이므로 아직 학습되지 않았음.
model = LinearRegression()

# fit() : 학습
model.fit(x_train, y_train) # 학습

# coef_ : 모델의 회귀계수 / 회귀식의 기울기 / 가중치
# ex) y = 2x + 3 일때 model.coef_는 [2.]
print('\n회귀 계수 : 회귀 식의 기울기')
print(model.coef_)

# intercept_ : 절편(y절편)
# ex) y = 2x + 3 일때 model.intercept_ 는 3.0
print('\n회귀식의 y 절편')
print(model.intercept_)

print('\n수식 표현')
print(f'y = {model.coef_[0]} * x + {model.intercept_}')
'''
이 모델은 독립 변수와 종속 변수 사이에 음의 상관 관계를 가지고 있음을 보여줍니다.
가중치가 -0.0076554은 독립 변수가 1증가할 때, 종속 변수는 대략 0.0076554씩 감소함을 의미합니다.
y 절편 46.60365052224634는 독립 변수의 값이 0일 때 예상되는 종속 변수의 값입니다.
'''

# score(테스트용 독립변수, 테스트용 종속변수) : R^2 결정 계수
# (모델의 설명력, 신뢰도를 나타내는 통계지표) -> 1에 가까울 수록 완벽한 모델
r_square = model.score(x_test, y_test)
print(f'R^2 결정 계수 : {r_square}')
'''
설명력 : 모델이 종속 변수의 변동 중에서 약 68.9%를 설명하고 있습니다.
즉, 독립 변수들이 종속 변수에 상당한 영향력을 미치고 있음을 의미합니다.

잔여 변동 : 나머지 31.1% 정도는 모델이 잘 설명하지 못하는 부분으로, 데이터의 불확실성,
혹은 모델의 구조적인 한계일 수 있습니다.
'''

# prediction은 머신 러닝이 학습한 모델을 이용하여 예측해준 정보
prediction = model.predict(x_test)

# 오차(잔차) 계산
test_preds = pd.DataFrame()
test_preds['label'] = y_test # 실제 정답 데이터를 의미하는 label
test_preds['prediction'] = prediction # 학습한 모델이 예측해준 정보
# squared_error : (실제정답 - 예측값)의 제곱 -> 0에 가까워져야 이상적임
# 공식
test_preds['squared_error'] = (test_preds['label'] - test_preds['prediction']) ** 2

print('\n평균 제곱 오차 데이터 프레임')
print(test_preds.head())

filename = 'test_preds.csv'
test_preds.to_csv(dataOut + filename, index=False)
print(f'{filename} 파일 생성됨')

# 평균 제곱 오차(mean squared error) -> 실제 데이터와 예측 데이터의 오차의 평균의 제곱
# 선형 회귀의 목적은 평균 제곱 오차 (MSE)를 최소로 하는 것.
mse = test_preds['squared_error'].mean()
print(mse)

print('산점도와 회귀선')
plt.figure(figsize=(10, 10))
# 머신러닝에서 정답을 label / 모델의 예측 : prediction
sns.regplot(x='label', y='prediction', data=test_preds, line_kws={'color':'red'})
plt.title('정답과 예측 값의 산점도 그래프')
plt.xlim([5, 45])
plt.ylim([5, 45])
filename = 'regplot.png'
plt.savefig(dataOut + filename)
print(filename + ' 파일이 저장되었습니다.')

# 여기서 부터 다중 회귀 분석입니다.
print('multidf.columns')
print(multidf.columns)

# 독립 변수와 종속 변수를 분리
independent_variable = ['cylinders', 'horsepower', 'weight']
x = multidf[independent_variable]
y = multidf['mpg']

# 훈련용 데이터와 테스트용 데이터 분리
x_train, x_test, y_train, y_test = \
    train_test_split(x, y, test_size=0.3, random_state=10)

print(f'훈련용 데이터 형상 : {x_train.shape}')
print(f'테스트용 데이터 형상 : {x_test.shape}')

# 다중 회귀 분석을 위한 모델 생성
multi_model = LinearRegression()

multi_model.fit(x_train, y_train)

r_square = multi_model.score(x_test, y_test)
print(f'R^2 결정 계수 : {r_square}')

slope = multi_model.coef_.tolist()
print(f'기울기 : {slope}')
print(f'y절편 : {multi_model.intercept_}')

# 막대 그래프 그리기
plt.bar(independent_variable, slope)
plt.title('회귀식의 기울기')
plt.xlabel('변수')
plt.ylabel('기울기')
plt.savefig(dataOut + 'independent_variable_slope.png')

prediction = multi_model.predict(x_test)

fig, axes = plt.subplots(1, 3, figsize=(12, 5))

for i, col in enumerate(x_test.columns):
    axes[i].plot(x_train[col], y_train, 'o', label='훈련 데이터')
    axes[i].plot(x_test[col], prediction, 'r+', label='예측값')
    axes[i].set_xlabel(col)
    axes[i].set_ylabel('mpg')
    # 범례 위치는 데이터의 분포를 보시고, 적정 위치에 두세요.
    axes[i].legend(loc='best')

plt.suptitle('독립 변수에 따른 종속 변수 산점도', size=15)
plt.savefig(dataOut + 'multivariance_regression.png')

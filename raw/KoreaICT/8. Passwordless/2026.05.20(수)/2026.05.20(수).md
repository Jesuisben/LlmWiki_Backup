\* 추가 사항
가상 머신의 할당된 용량을 바꾸면 어떻게 변할까?
가상 머신의 할당된 용량을 바꾸고 아래의 명령어를 입력

- 명령어들
```shell
# 1. Check the current disk layout
su -
lsblk -o NAME,TYPE,SIZE,FSTYPE,MOUNTPOINT
# 2. Extend the partition to use the free space
# (Ex: when partition 2 is an LVM2_member)
parted /dev/nvme0n1 resizepart 2 100%
partprobe
# 3. Resize the LVM PV*
pvresize /dev/nvme0n1p2
# 4. Extend the root LV** and filesystem
lvextend -r -l +100%FREE /dev/mapper/rl-root
# 5. Verify the result
df -h /
# 6. Restart the GIGA service
systemctl restart giga.service
```
PV\* : Physical Volume
LV\*\* : Logical Volume

- 다시 웹페이지 확인
용량이 바뀌어 있음

\* 주의 : VMware는 할당된 용량을 크게만 수정가능하고 작게는 수정이 안됨

#### \# DataFrame 추가 공부 내용
##### DataFrame 개념
label : index나 columns의 실제 값
position : index나 columns의 위치 값 (무조건 0부터 시작함)
행 : index
열 : columns
선택자(key, 인덱서 등) : \[\]안에 들어가는 데이터들 (메소드의 ()안에 들어가는 것을 매개변수라고 부르는 것과 같음)

- 기본 데이터 준비
```python
myframe = pd.DataFrame(
    [
        [10, 20, 30],
        [40, 50, 60],
        [70, 80, 90]
    ],
    index=['행1', '행2', '행3'],
    columns=['열1', '열2', '열3']
)
'''
출력 결과 (DataFrame)
    열1  열2  열3
행1  10  20  30
행2  40  50  60
행3  70  80  90
'''
```


##### 1. myframe\[\]
###### 1) myframe\[\] 인덱싱
선택자는 DataFrame의 열(columns)의 label이 기준임

- myframe\["열의 label"\]
해당 열에 해당하는 모든 행을 가져오고 Series로 반환
```python
myframe['열3'] # 사실상 같음 myframe.loc[:, '열3']
'''
행1    30
행2    60
행3    90
Name: 열3, dtype: int64
'''
```

- myframe\[\["열의 label"\]\]
해당 열에 해당하는 모든 행을 가져오고 DataFrame으로 반환
선택자로 단순히 "열의 label"을 사용하는 것이 아니라 리스트 형태로 "열의 label"를 가져온 것이라
기본적으로 여러개가 선택자로 올때의 규칙이 적용되어 1개여도 DataFrame 형태로 반환함
-> 여러개의 열을 선택하면 자동적으로 Series로는 표현할 수가 없고 DataFrame으로 표현이 가능함
```python
myframe[['열3']]
'''
     열3
행1  30
행2  60
행3  90
'''
```

- myframe\[\["열의 label01", "열의 label02"\]\]
해당 열에 해당하는 모든 행을 가져오고 DataFrame으로 반환
```python
myframe[['열2', '열3']]
'''
    열2  열3
행1  20  30
행2  50  60
행3  80  90
'''
```


###### 2) myframe\[\] 슬라이싱
선택자는 DataFrame의 행(index)이 기준임

- myframe\[행의_position : 행의_position : 행의_position\]
해당 행의 범위(이상~미만)의 행들과 그에 해당하는 열들을 가지고 DataFrame으로 반환
```python
myframe[0:2] # 사실상 myframe.iloc[0:2]와 같음
'''
    열1  열2  열3
행1  10  20  30
행2  40  50  60
'''
```


###### 3) myframe\[\] 불리언 Series 인덱싱
선택자는 행들로 이루어진 Series임
해당 행들 중에서 value가 True 인것들만 고르고 그에 해당하는 열들을 가지고 DataFrame으로 반환
```python
myframe['열1'] > 20
'''
행1    False
행2     True
행3     True
Name: 열1, dtype: bool
'''

myframe[myframe['열1'] > 20] # 행1은 False라서 제외됨
'''
    열1  열2  열3
행2  40  50  60
행3  70  80  90
'''
```



##### 2. myframe.loc\[\]
loc은 index와 columns의 label을 기준으로 데이터를 선택함
행이나 열을 적지 않으면 기본값으로 ":" 이 설정됨
즉 모든 행이나 열을 선택하게 됨

- 선택자에 따른 반환 타입
```text
1. 행과 열을 모두 단일 label로 선택
  → 스칼라 반환

2. 행과 열 중 한 축은 단일 label로 선택하고,
  다른 축은 리스트 또는 슬라이싱으로 선택
  → Series 반환

3. 행과 열을 모두 리스트 또는 슬라이싱으로 선택
  → DataFrame 반환
```

###### 1) myframe.loc\["행의 label", "열의 label"\]
- myframe.loc\["행의 label", "열의 label"\]
해당 행과 해당 열에 해당하는 데이터(스칼라)를 반환함
```python
myframe.loc["행1", "열1"]
'''
10
'''
```

- myframe.loc\["행의 label"\]
해당 행에 해당하는 모든 열을 가져와서 Series로 반환
Series로 나오는 이유는 선택된 행은 하나이지만 선택된 열은 리스트(복수) 형식이기 때문에
하나의 데이터(스칼라)로 표현할 수 없고 DataFrame으로 표현할 수도 없어서임
```python
myframe.loc["행1"] # myframe.loc["행1", :] 굳이 따지자면 이것과 같음
'''
열1    10
열2    20
열3    30
Name: 행1, dtype: int64
'''
```

- myframe.loc\[:, "열의 label"\]
해당 열에 해당하는 모든 행을 가져와서 Series로 반환
```python
myframe.loc[:, "열1"] # myframe["열1"] 과 같음
'''
행1    10
행2    40
행3    70
Name: 열1, dtype: int64
'''
```


###### 2) myframe.loc\[\["행의 label"\], \["열의 label"\]\]
- myframe.loc\[\["행의 label"\]\]
행의 label 선택자 위치에 리스트 형식이 오고, 열의 label 선택자 위치에 아무것도 적지 않아
기본값인 ":"(모든)이 와서 해당 행에 해당하는 모든 열을 가져와서 DataFrame으로 반환함
\["행의 label01", "행의 label02"\]을 넣어도 비슷하게 DataFrame으로 반환함
```python
# myframe.loc[["행1"], :]와 같은데 :은 사실 [모든 열]이니까
# 사실 myframe.loc[["행1"], ["열1", "열2", "열3"]]과 같음
myframe.loc[["행1"]]
'''
    열1  열2  열3
행1  10  20  30
'''

myframe.loc[["행1", "행2"]]
'''
    열1  열2  열3
행1  10  20  30
행2  40  50  60
'''
```

- myframe.loc\[:, \["열의 label"\]\]
위의 myframe.loc\[\["행의 label"\]\]와 똑같음
```python
# 사실 myframe.loc[["행1", "행2", "행3"], ["열1"]]과 같음
myframe.loc[:, ["열1"]]
'''
    열1
행1  10
행2  40
행3  70
'''

myframe.loc[:, ["열1", "열2"]]
'''
    열1  열2
행1  10  20
행2  40  50
행3  70  80
'''
```


###### 3) myframe.loc\["행의 label 슬라이싱", "열의 label 슬라이싱"\]
- myframe.loc\["행의 label 슬라이싱", "열의 label 슬라이싱"\]
label을 이용한 슬라이싱은 시작 label과 끝 label을 모두 포함함
```python
# 행의 행1~행2까지 / 열은 열2~열3까지
# 둘다 리스트 형식이라 DataFrame으로 반환함
myframe.loc['행1':'행2', '열2':'열3']
'''
    열2  열3
행1  20  30
행2  50  60
'''
```



##### 3. myframe.iloc\[\]
myframe.loc\[\]과 전체 구조, 반환 규칙이 거의 같고 다소 차이점이 존재함

- 차이점
```text
1. label 대신 position 사용
myframe.loc[행 label 선택자, 열 label 선택자]
myframe.iloc[행 position 선택자, 열 position 선택자]
->
myframe.loc["행1", "열3"]
myframe.iloc[0, 2]

2. 슬라이싱의 끝 position은 포함하지 않음
myframe.loc["행1":"행2", "열1":"열2"] # 행2와 열2까지 포함
myframe.iloc[0:2, 0:2] # 행 position 2와 열 position 2는 미포함

3. 음수 position 사용 가능
myframe.iloc[-1] # 마지막 행
myframe.iloc[:, -1] # 마지막 열
```

- 선택자에 따른 반환 타입
```text
1. 행과 열을 모두 단일 position 선택
  → 스칼라 반환

2. 행과 열 중 한 축은 단일 position으로 선택하고,
  다른 축은 리스트 또는 슬라이싱으로 선택
  → Series 반환

3. 행과 열을 모두 리스트 또는 슬라이싱으로 선택
  → DataFrame 반환
```


###### 1) myframe.iloc\[행의 position, 열의 position\]
- myframe.iloc\[행의 position, 열의 position\]
해당 position의 행과 해당 position의 열에 해당하는 데이터(스칼라)를 반환함
```python
myframe.iloc[0, 0]
'''
10
'''
```

- myframe.iloc\[행의 position\]
해당 position의 행에 해당하는 모든 열을 가져와서 Series로 반환
Series로 나오는 이유는 선택된 행은 하나이지만 선택된 열은 리스트(복수) 형식이기 때문에
하나의 데이터(스칼라)로 표현할 수 없고 DataFrame으로 표현할 수도 없어서임
```python
myframe.iloc[0] # myframe.iloc[0, :] 굳이 따지자면 이것과 같음
'''
열1    10
열2    20
열3    30
Name: 행1, dtype: int64
'''
```

- myframe.iloc\[:, 열의 position\]
해당 position의 열에 해당하는 모든 행을 가져와서 Series로 반환
```python
myframe.iloc[:, 0] # myframe["열1"]과 결과가 같음
'''
행1    10
행2    40
행3    70
Name: 열1, dtype: int64
'''
```


###### 2) myframe.iloc\[\[행의 position\], \[열의 position\]\]
- myframe.iloc\[\[행의 position\]\]
행의 position 선택자 위치에 리스트 형식이 오고, 열의 position 선택자 위치에 아무것도 적지 않아
기본값인 ":"(모든)이 와서 해당 행에 해당하는 모든 열을 가져와서 DataFrame으로 반환함
\[행의 position01, 행의 position02\]를 넣어도 비슷하게 DataFrame으로 반환함
```python
# myframe.iloc[[0], :]와 같은데 :은 사실 [모든 열]이니까
# 사실 myframe.iloc[[0], [0, 1, 2]]와 같음
myframe.iloc[[0]]
'''
    열1  열2  열3
행1  10  20  30
'''

myframe.iloc[[0, 1]]
'''
    열1  열2  열3
행1  10  20  30
행2  40  50  60
'''
```

- myframe.iloc\[:, \[열의 position\]\]
위의 myframe.iloc\[\[행의 position\]\]와 똑같음
```python
# 사실 myframe.iloc[[0, 1, 2], [0]]과 같음
myframe.iloc[:, [0]]
'''
    열1
행1  10
행2  40
행3  70
'''

myframe.iloc[:, [0, 1]]
'''
    열1  열2
행1  10  20
행2  40  50
행3  70  80
'''
```

- myframe.iloc\[\[행의 position\], \[열의 position\]\]
행의 position 선택자와 열의 position 선택자에 모두 리스트 형식이 오면
선택된 행과 열을 가지고 DataFrame으로 반환함
```python
myframe.iloc[[0], [0]]
'''
    열1
행1  10
'''

myframe.iloc[[0, 1], [1, 2]]
'''
    열2  열3
행1  20  30
행2  50  60
'''
```


###### 3) myframe.iloc\[행의 position 슬라이싱, 열의 position 슬라이싱\]
- myframe.iloc\[행의 position 슬라이싱, 열의 position 슬라이싱\]
position을 이용한 슬라이싱은 시작 position은 포함하지만 끝 position은 포함하지 않음
```python
# 행은 position 0 이상 2 미만 / 열은 position 1 이상 3 미만
# 둘 다 슬라이싱 형식이라 DataFrame으로 반환함
myframe.iloc[0:2, 1:3]
'''
    열2  열3
행1  20  30
행2  50  60
'''
```

- myframe.iloc\[행의 position 슬라이싱\]
행의 position만 슬라이싱하고 열 선택자 위치에 아무것도 적지 않아
기본값인 ":"(모든)이 와서 해당 범위의 행과 모든 열을 가져와서 DataFrame으로 반환함
```python
myframe.iloc[0:2] # myframe.iloc[0:2, :]와 같음
'''
    열1  열2  열3
행1  10  20  30
행2  40  50  60
'''
```

- myframe.iloc\[:, 열의 position 슬라이싱\]
열의 position만 슬라이싱하려면 행 선택자 위치에 ":"을 작성해야 함
해당 범위의 열과 모든 행을 가져와서 DataFrame으로 반환함
```python
myframe.iloc[:, 0:2]
'''
    열1  열2
행1  10  20
행2  40  50
행3  70  80
'''
```

- myframe.iloc\[행의 position : 행의 position : 간격, 열의 position : 열의 position : 간격\]
position 슬라이싱에는 시작 position과 끝 position 외에 간격을 지정할 수도 있음
```python
# 행은 position 0 이상 3 미만의 범위에서 2칸 간격으로 선택
# 열은 position 0 이상 3 미만의 범위에서 2칸 간격으로 선택
myframe.iloc[0:3:2, 0:3:2]
'''
    열1  열3
행1  10  30
행3  70  90
'''
```


###### 4) myframe.iloc\[-position\]
음수 position을 사용하면 행이나 열을 뒤에서부터 선택할 수 있음
position이 -1이면 마지막 위치를 의미함

- myframe.iloc\[-1\]
마지막 행에 해당하는 모든 열을 가져와서 Series로 반환
```python
myframe.iloc[-1]
'''
열1    70
열2    80
열3    90
Name: 행3, dtype: int64
'''
```

- myframe.iloc\[:, -1\]
마지막 열에 해당하는 모든 행을 가져와서 Series로 반환
```python
myframe.iloc[:, -1]
'''
행1    30
행2    60
행3    90
Name: 열3, dtype: int64
'''
```


음수 position을 이용한 슬라이싱도 가능함
- myframe.iloc\[-2:\]
뒤에서 두 번째 행부터 마지막 행까지 가져와서 DataFrame으로 반환
```python
myframe.iloc[-2:]
'''
    열1  열2  열3
행2  40  50  60
행3  70  80  90
'''
```

- myframe.iloc\[:, -2:\]
뒤에서 두 번째 열부터 마지막 열까지 가져와서 DataFrame으로 반환
```python
myframe.iloc[:, -2:]
'''
    열2  열3
행1  20  30
행2  50  60
행3  80  90
'''
```

import pandas as pd

# DataFrame에 들어갈 index와 columns 준비

# mylist02의 25개 값을 5행 5열의 2차원 NumPy 배열로 재배열한 뒤,

| 심병식 | 160 | 170 | 180 | 190 | 200 |

```python
# DataFrame에서 찾기
# 열 색인 : 판다스의_DataFrame_객체["열 색인 이름"]
# 행 색인 : 판다스의_DataFrame_객체.iloc[position] / 판다스의_DataFrame_객체.loc["label"]

# position으로 찾기
# 행으로 찾기  (열로 찾을때는 그냥 함수없이 판다스의_DataFrame_객체.["열 색인 이름"] 하면 됨)
# 문법 : 판다스의_DataFrame_객체.iloc[행, 열]
# 매개변수를 한개만 입력하면 행으로 인식함
# 열로 검색시 행에 대한 값도 적어줘야 함 ( : 기호는 모두를 뜻하는 기호)
myframe02.iloc[1] # 1번 행 색인
```
(출력)
중구      60
용산구     70
마포구     80
노원구     90
은평구    100

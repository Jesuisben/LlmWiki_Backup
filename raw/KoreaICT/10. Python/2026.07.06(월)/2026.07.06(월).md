1) 샘플 URL
```text
http://openapi.seoul.go.kr:8088/(인증키)/xml/SeoulPublicLibraryInfo/1/5/
```

1-1) 풀이
http://openapi.seoul.go.kr:8088/ : 고정
(인증키) : 내가 발급받은 인증키
/xml/SeoulPublicLibraryInfo : 내가 선택한 서울시 공공도서관 현황정보에 관한 URL
/1/5/ : 1페이지부터 5페이지까지의 정보

2) 요청인자
	파라미터들 + URL에 있는 내용들

| 변수명         | 타입          | 변수설명   | 값설명                               |
| ----------- | ----------- | ------ | --------------------------------- |
| KEY         | String(필수)  | 인증키    | OpenAPI 에서 발급된 인증키                |
| TYPE        | String(필수)  | 요청파일타입 | xml : xml<br>json파일 : json        |
| SERVICE     | String(필수)  | 서비스명   | tbCycleRentUseDayInfo             |
| START_INDEX | INTEGER(필수) | 요청시작위치 | 정수 입력 (페이징 시작번호 입니다 : 데이터 행 시작번호) |
| END_INDEX   | INTEGER(필수) | 요청종료위치 | 정수 입력 (페이징 끝번호 입니다 : 데이터 행 끝번호)   |
| RENT_DT     | STRING(필수)  | 대여일자   | YYYYMMDD                          |

3) 출력값
	반환해주는 값들

| No  | 출력명              | 출력설명                   |
| --- | ---------------- | ---------------------- |
| 공통  | list_total_count | 총 데이터 건수 (정상조회 시 출력됨)  |
| 공통  | RESULT.CODE      | 요청결과 코드 (하단 메세지설명 참고)  |
| 공통  | RESULT.MESSAGE   | 요청결과 메시지 (하단 메세지설명 참고) |
| 1   | RENT_DT          | 대여일자  <br>(YYYYMMDD)   |
| 2   | RENT_ID          | 대여소번호                  |
| 3   | RENT_NM          | 대여소명                   |
| 4   | RENT_TYPE        | 대여구분코드                 |
| 5   | GENDER_CD        | 성별                     |
| 6   | AGE_TYPE         | 연령                     |
| 7   | USE_CNT          | 대여건수                   |
| 8   | EXER_AMT         | 운동량                    |
| 9   | CARBON_AMT       | 탄소절감량                  |
| 10  | MOVE_METER       | 이동거리(M)                |
| 11  | MOVE_TIME        | 이용시간(분)                |

6) bicycle01.py 코드 완성

- 코드 내용
```python
import os  
# 0부터 시작해서 무한히 증가하는 숫자를 생성하기 위한 패키지 호출  
# (페이지 수가 몇 페이지까지인지 모를 때, break를 만날 때까지 계속 숫자를 올리기 위해 사용)  
import itertools  
  
# 사이트에서 정보를 요청해서 가져오는 패키지 호출  
import requests  
  
# 데이터 분석을 위한 pandas 패키지 호출  
# DataFrame 만들기 위한 함수 가져오는 패키지  
import pandas as pd  
  
# .env 파일에 저장된 환경변수를 로드하기 위한 함수(load_dotenv) 호출  
from dotenv import load_dotenv  
  
# .env 파일의 내용을 환경변수로 등록  
load_dotenv()  
  
# 등록된 환경변수에서 서울시 API 키 가져오기  
access_token = os.getenv("SEOUL_API_KEY")  
# print(access_token)  
  
# API 키가 제대로 로드되지 않았다면 프로그램 중단 및 에러 메시지 출력  
if not access_token:  
    raise ValueError("관련 API 설정 정보가 존재하지 않습니다.")  
  
# 한 페이지당 가져올 데이터 개수  
page_size = 1000  
  
# 모든 자전거 데이터를 담을 빈 리스트 생성  
total_bicycle = list()  
  
# itertools.count() : 조건 없이 0부터 무한대까지 숫자를 계속 생성해 주는 역할  
# page_number를 0부터 1씩 무한히 증가시키며 반복 시작  
for page_number in itertools.count():  
    print(f"{page_number + 1}번째 페이지 추출중입니다.")  
  
    # 가져올 페이지의 번호의 시작점  
    begin_row = str(page_number * page_size + 1)  
    # 가져올 페이지의 번호의 끝점  
    end_row = str((page_number + 1) * page_size)  
  
    # 고정값은 변수로 만들어놓음  
    url = "http://openapi.seoul.go.kr:8088/"  
    url += access_token  
    url += "/json/bikeList/"  
    url += begin_row + "/"  
    url += end_row + "/"  
    # print(url)  
  
    # 해당 url에서 정보들을 요청해서 가져오는 패키지와 함수  
    response = requests.get(url)  
    # json() 함수로 json 형식으로 변환  
    json_data = response.json()  
    # print(json_data)  
  
    try:  
        # 실제 URL에 들어가서 보거나 샘플 URL의 예제를 보고 키 값을 찾아서 입력하기  
        rent_bike_list = json_data["rentBikeStatus"]["row"]  
        print(len(rent_bike_list))  
  
        for one_rent in rent_bike_list:  
            total_bicycle.append(one_rent)  
  
        # (테스트용) 일정 조건시 강제 종료  
        if page_number == 2:  
            break  
  
    except Exception as err:  
        print("더 이상 데이터가 없어서 오류가 발생합니다.")  
        print(f"오류 내용 : {err}")  
        break  
    # end try  
  
print("데이터 수집 끝")  
  
# 수집한 데이터를 CSV파일로 만들어서 저장하기  
dataIn = "../dataIn/"  
filename = dataIn + "bikeList.csv"  
  
# 수집한 데이터를 DataFrame에 넣기  
bike_frame = pd.DataFrame(total_bicycle)  
# DataFrame을 csv로 만들기  
bike_frame.to_csv(filename)  
print(f"{filename}이 저장되었습니다.")
```

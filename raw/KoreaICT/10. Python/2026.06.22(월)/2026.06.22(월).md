#### \# range 함수 (Python(이론).pdf (P.51))
연속적인 정수 데이터를 뽑아낼때 사용하는 함수

- 용법
1) range(stop) : 0부터 시작하여 stop-1까지의 연속된 숫자를 생성
ex) range(5) : 0부터 4까지의 숫자를 생성

2) range(start, stop) : start부터 시작하여 stop-1까지의 연속된 숫자를 생성
ex) range(1, 5) : 1부터 4까지의 숫자를 생성

3) range(start, stop, step) : start부터 시작하여 stop-1까지의 숫자를 step 간격으로 생성
ex) range(1, 10, 2) : 1부터 9까지의 숫자를 2씩 증가시키면서 생성

- 실제 사용
```python
for i(지역변수) in range(용법대로) :
	print(i)
```


\* 범위 객체를 리스트로 변환하여 사용하기
```python
list_of_numbers = list(range(5))
print(list_of_numbers) # 출력: [0, 1, 2, 3, 4]
```

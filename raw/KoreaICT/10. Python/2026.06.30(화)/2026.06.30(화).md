#### \# pandas 관련 (https://pandas.pydata.org/docs/reference/index.html)
- Series
1) 개념
1차원 자료구조
value와 index가 1:1로 대응되는 자료구조
오라클 테이블의 한 개의 행(row) 또는 한 개의 열(column)과 비슷한 개념
여러 개의 Series가 모여 DataFrame을 구성함

2) 구성요소
position : 데이터를 저장하는 내부 위치(0부터 시작) -> 리스트의 인덱스같은 위치 정보 (데이터가 아님)
index : 데이터를 식별하기 위한 이름(라벨) -> 각 value와 1:1로 대응되는 인덱스(식별자)(라벨)
value : 실제 저장된 데이터
```text
* 중요 *
리스트에서도 position과 index는 개념적으로는 다르다.
다만 리스트에서는 position을 나타내는 index가 항상 일치하므로
보통 index라는 용어만 사용한다.

Series에서는 position과 index가 서로 독립적으로 존재하므로
둘을 구분해서 사용한다.
```

3) 생성자와 속성들 공부
Constructor : 생성자
Attributes : 속성
Conversion : 형변환
Indexing : 인덱싱, 슬라이싱
iteration : 반복 가능한 집합
등등


- DataFrame
여러 개의 Series가 모인 2차원 자료구조
오라클의 테이블(Table) 과 비슷한 개념
행(row)과 열(column)으로 데이터를 관리함

```python
# range(5, 8) : 숫자의 범위를 기억하는 range 객체 / 아직 리스트처럼 숫자를 저장한게 아님
myrange = range(5, 8)
print(myrange)

# pandas의 Series 객체 생성
# pd.Series(data=None, index=None, dtype=None, name=None, copy=False) -> pandas.Series
# 1. data가 iterable 객체이면 요소를 하나씩 순회(iterate)하여 Series의 value로 사용함
# 사전(dict)은 key를 index로, value를 value로 사용함
# 내부적으로 반복문(for)으로 하나씩 꺼내는 것과 동일하게 이해하면 됨
# index를 지정하지 않으면 0부터 자동 생성됨
# 2. data가 iterable이 아니고 scalar(int, float, bool 등)인 경우
# index를 지정하지 않으면 값 하나를 가진 Series가 생성됨 (인덱스는 0임)
# index를 지정하면 index의 개수만큼 같은 값이 모든 index에 채워짐
# ex) pd.Series("A", index=[1,2,3,4]) 이 Series 객체는 index가 1, 2, 3, 4 가 있고 모든 인덱스의 value는 문자열 A임
myseries01 = pd.Series(myrange)
myseries01
```

- 사전(dict)으로 Series 생성
- 파이썬 사전 이용하기
```python
# 1. 사전(dict)을 Series()의 data 매개변수에 넣으면
# key는 Series의 index가 되고 value는 Series의 value가 됨
# index를 따로 지정하지 않아도 사전의 key가 그대로 index로 사용됨

# 2. Series()의 index 매개변수를 따로 지정하면
# 지정한 index와 사전의 key를 비교하여 일치하는 key의 value를 가져옴
# 사전에 존재하지 않는 index를 지정하면 해당 value는 NaN이 됨
# ex) pd.Series(sdata, index=['영국', '한국', '일본'])
# 영국은 사전의 key와 일치하므로 2000을 가져옴
# 한국은 사전에 존재하지 않으므로 NaN이 들어감
# 일본은 사전의 key와 일치하므로 3000을 가져옴
# NaN이 포함되면서 Series 전체가 float64가 되어
# 실제 출력은 2000.0, NaN, 3000.0으로 표시됨
sdata = {'일본': 3000, '영국': 2000}
myseries01 = pd.Series(sdata)
myseries01
```
(출력)

영국   2000
dtype: int64

- 문자열 lable 사용
```python
# 라벨(lable) : index에 저장된 실제 값
# 위치(position) : index안의 데이터의 위치값
# 값(value) : Series의 값
# scalar (일반적인 int, float, str 값)로 반환하는 방법은 인덱싱밖에 없음
# Series 객체를 인덱싱이나 슬라이싱할때는 기본적으로는 []안에 index가 들어가야함

```
(출력)

60

```python
# 1. 리스트 타입의 데이터를 []에 넣어 인덱싱하면

# 선택한 index와 그에 대응하는 value를 포함한 새로운 Series를 반환함
# 2. 개념적으로는 다음과 같이 이해하면 됨.
# for idx in ['미국']:

# 3. 개념적으로는 리스트의 요소를 하나씩 순회하면서
# 해당 index와 value를 선택한다고 이해하면 됨
print('색인의 이름으로 값 읽기(Series로 반환)')
```

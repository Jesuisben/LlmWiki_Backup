인증키 발급 받는 방법

---------------------------------------------------------------------

OpenAI API를 사용하기 위해 필요한 인증키(API Key) 발급, 결제 카드 등록, 토큰 사용량 확인 방법을 순서대로 설명드리겠습니다.

1. 인증키(API Key) 발급 방법
① OpenAI Platform 접속
OpenAI Platform(https://platform.openai.com/) 사이트에 로그인합니다.

② 로그인
ChatGPT 계정과 동일한 계정으로 로그인합니다.

③ API Keys 메뉴 선택
https://platform.openai.com/home

왼쪽 메뉴에서 API Keys를 클릭합니다.
또는
Dashboard → API Keys로 이동합니다.

④ Create new secret key 클릭
우측 상단의 [+ Create new secret key] 버튼을 클릭합니다.

⑤ 키 이름, 프로젝트 이름을 입력합니다.
예시) Python, TravelAgent, MyProject 

⑥ Secret Key 복사
생성되는 키는 한 번만 표시되므로 반드시 복사하여 안전한 곳에 보관해야 합니다.
Python에서는 일반적으로 .env 파일에 저장합니다.

.env 파일에 저장 예시)
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxx


2. 카드 등록 방법
OpenAI API는 사용량 기반 과금이므로 결제 수단을 등록해야 합니다.
하단의 "Go to Billing" 버튼을 클릭해도 됩니다
사이트 : https://platform.openai.com/settings/organization/billing/overview

Billing 상단 메뉴에서 Overview Tab을 클릭하고 "Add payment details"버튼을 클릭합니다.
카드 정보를 입력하고, 저장하면 등록이 완료됩니다.
참고로 체크카드도 가능한 경우가 있지만 가장 안정적인 것은 VISA/MASTER/AMEX/신용카드입니다.


3. 사용 한도(Usage Limit) 설정
Billing 메뉴에서 [Usage limits]를 선택합니다.

예를 들어
Monthly budget 20 USD처럼 설정하면 한 달 사용량이 20달러에 도달하면 API 사용이 중지됩니다.

5. Python 코드에서 토큰 사용량 확인

응답 객체에는 사용량 정보가 포함됩니다.

예시:

from openai import OpenAI

client = OpenAI()

response = client.responses.create(
    model="gpt-5",
    input="안녕하세요."
)

print(response.usage)

예상 출력:

{
    "input_tokens": 12,
    "output_tokens": 18,
    "total_tokens": 30
}

또는 필요한 값만 출력할 수 있습니다.

print(response.usage.input_tokens)
print(response.usage.output_tokens)
print(response.usage.total_tokens)
6. 지금까지 사용한 전체 비용 확인

Platform의 Usage 페이지에서는 다음과 같은 항목도 확인할 수 있습니다.

오늘 사용량
이번 달 사용량
프로젝트별 사용량
모델별 사용량
API Key별 사용량(설정에 따라)
7. 429 오류(Quota Exceeded)가 발생하는 경우

다음과 같은 오류:

HTTPError: 429
insufficient_quota

는 일반적으로 다음 원인 중 하나입니다.

결제 수단이 등록되지 않음
무료 크레딧이 모두 소진됨
월 사용 한도에 도달함
프로젝트의 예산 제한이 설정되어 있음

이 경우에는 Billing과 Usage 페이지에서 결제 수단, 예산 한도, 현재 사용량을 확인해 보세요.

요약
항목	확인 위치
API 인증키 발급	OpenAI Platform → API Keys
카드 등록	OpenAI Platform → Billing → Payment methods
월 예산 설정	OpenAI Platform → Billing → Usage limits
토큰 사용량 확인	OpenAI Platform → Usage
API 응답별 토큰 확인	response.usage 객체

이전에 API 사용 중 429 오류와 API Key는 정상 동작하는데 대시보드가 헷갈린다는 상황을 말씀해 주셨던 만큼, 원하시면 현재 OpenAI Platform 화면을 기준으로 **
로그인부터 API Key 생성, 카드 등록, Usage 확인까지 스크린샷 형태로 따라할 수 있는 단계별 가이드**도 자세하게 만들어 드리겠습니다.

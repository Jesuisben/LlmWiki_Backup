#### \# IP주소 관련 (네트워크 서비스) (cloud.03.AWS 교안(이론).pdf(P.84~86))
- 옥텟
192.168.1.10
192가 들어간 공간 하나를 옥텟이라고 부름
-\> IP 주소는 옥텟(8)이 4개임 (콤마는 3개)

- IP주소 비교
IP 주소(10진법) : 255.255.255.0
IP 주소(2진법) : 11111111.11111111.11111111.00000000

- Network 주소와 Host 주소
IP주소 :
Network 주소 : 네트워크 아이디(Network ID) : 어느 네트워크에 속했는지
-\>
Host 주소 : 호스트 아이디(Host ID) : 그 네트워크 안에서의 개별 장치 번호 (식별을 위한)
-\>

- IP CIDR (cloud.03.AWS 교안(이론).pdf(P.87))
1\. Case 01
IP 주소 : 10.250.1.0/24
Network 주소 : 10.250.1
Host 주소 : 0
Subnet mask 길이 : 24 -\> 24는 24비트를 뜻하고 24는 8x3이고 옥텍 한개에 들어갈 수 있는 숫자의 수는 2의 8승(8비트) 총 3개의 옥텍을 Network 주소로 사용하는 것
-\> 앞쪽 24비트는 무조건 고정

0~255의 숫자는 2의 8승인 256개

2\. Case 02
IP 주소 : 10.250.1.0/24
Network 주소 : 10.250
Host 주소 : 0.0
Subnet mask 길이 : 16 -\> 16는 16비트를 뜻하고 16는 8x2이고 옥텍(8비트) 한개에 들어갈 수 있는 숫자의 수는 2의 8승(8비트) 총 2개의 옥텍을 Network 주소로 사용하는 것
-\> 앞쪽 16비트는 무조건 고정

0~255의 숫자는 2의 8승인 256개

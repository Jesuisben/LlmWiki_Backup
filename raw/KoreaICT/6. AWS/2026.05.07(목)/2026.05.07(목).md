(렌탈이 아니고 내가 직접 만드는거)

- 수식들?

1) Region = Σ VPC_i (i=1 to n)


하나의 리전(Region) 안에는 여러 개의 VPC가 존재할 수 있다는 뜻
리전 : 서울, 도쿄처럼 물리적인 거대 지역
사용자(개발자)는 이 리전 안에 본인만의 가상 네트워크인 VPC를 여러 개 만들 수 있음

2) VPC    = Σ AZ_i (i=1 to n)


하나의 VPC는 여러 개의 AZ(가용 영역)에 걸쳐서 구성된다는 뜻
VPC는 논리적인 네트워크라 리전 전체를 커버함
하지만 실제 기계(서버)가 돌아가는 물리적 장소인 AZ는 보통 한 리전 안에 2~4개 정도 있는데,
하나의 VPC는 이 AZ들을 모두 포함하여 설계할 수 있음

3) AZ     = Σ Subnet_i (i=1 to n)


하나의 AZ(가용 영역) 안에는 여러 개의 서브넷(Subnet)이 존재한다는 뜻
AZ라는 큰 데이터 센터 묶음 안에서,
용도에 따라(예: 외부용, 내부 데이터베이스용) 네트워크를 잘게 쪼갠 것이 서브넷임
우리나라는 4개의 AZ가 있음 (2A / 2B / 2C / 2D - 2A와 2C를 주로 사용 / 나머지는 비용 발생?)

VPC - 탄력적 IP

6. 키페어 삭제

삭제 할 키페어 체크 -> 작업 -> 삭제

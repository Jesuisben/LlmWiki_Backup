# 2026-06-22 Python 제어문과 컬렉션

## 한 줄 요약

`if`/`elif`/`else`, `for`, `while`, `range`로 실행 흐름을 제어하고, `list`와 `tuple`을 이용해 여러 값을 묶어 처리하는 방법을 배웠다.

## 배운 내용

단순 if, 양자택일 if/else, 다중 조건 elif, `range(stop/start/step)`, `for`, `while True`와 `break`, list의 `append`/`insert`/`remove`/`count`/`index`, tuple의 불변성, `Counter`, `split()`/`join()`을 다뤘다.

## 실습 / 예제

학점 판정, 1부터 n까지 합계, 커피 재고가 0이 될 때까지 판매하는 프로그램, 커피 리스트와 빵 튜플 처리 문제를 풀었다.

## 헷갈린 점 / 질문

`range(1, 5)`는 1~4다. `list`는 수정 가능하지만 `tuple`은 직접 수정할 수 없다. `while True`는 종료 조건과 `break`가 없으면 끝나지 않는다.

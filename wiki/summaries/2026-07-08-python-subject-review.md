# Python 총정리

## 한 줄 요약
Python은 기본 문법과 객체지향을 익힌 뒤 파일·정규표현식·XML/JSON으로 외부 데이터를 다루고, Pandas·API·크롤링·지도·텍스트 마이닝으로 실제 데이터를 수집·분석·표현한 과목이다.

## 학습 흐름
기본문법·컬렉션 → 함수·모듈·객체지향 → 예외·파일 → 정규표현식·XML·JSON → Series·DataFrame → 조회·결측치·CSV → `concat`/`merge`/`pivot` → `groupby`/범주화/그래프 → Open API·크롤링·지오코딩·Folium → 한국어 텍스트 마이닝 순으로 확장됐다.

## 핵심 연결
초반의 list·dict·함수·예외 처리는 JSON 응답과 CSV 행을 처리하는 기반이 됐다. Oracle의 JOIN/GROUP BY 감각은 Pandas `merge`/`groupby` 이해에 연결됐지만, DataFrame은 DB 테이블이 아니라 Python 메모리 안의 객체다. 마지막 실습은 데이터를 가져오는 것에서 멈추지 않고 정제·결합·집계·시각화까지 이어지는 분석 파이프라인을 보여 준다.

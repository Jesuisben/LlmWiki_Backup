# 2026-03-12 Java 추상 클래스, 인터페이스, static/final

## 한 줄 요약

확장 for, 추상 메서드, 인터페이스, static/final을 통해 객체지향 설계를 확장한 날이다.

## 배운 내용

- 향상된 for문은 배열/컬렉션의 모든 요소를 순서대로 꺼낼 때 사용한다.
- 추상 클래스와 추상 메서드는 공통 틀은 제공하되 구체 동작은 하위 클래스가 구현하게 만든다.
- 인터페이스는 강제 표준 규범을 제공하며 Java에서 다중 구현 형태로 활용된다.
- `static`, `final`, `private static` 변수로 전체 객체가 공유하는 값과 바뀌지 않는 값을 다뤘다.

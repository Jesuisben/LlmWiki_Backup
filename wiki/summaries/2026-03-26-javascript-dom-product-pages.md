# 2026-03-26 JavaScript DOM과 상품 목록/상세 페이지

## 한 줄 요약

JavaScript 배열, DOM 조작, 이벤트를 이용해 상품 목록/상세 페이지를 만든 날이다.

## 배운 내용

- 상품 배열과 사용자 role 데이터를 만들어 관리자 전용 버튼 표시 로직을 구현했다.
- `forEach`, 템플릿 문자열, 화살표 함수로 배열 데이터를 HTML 카드 목록으로 렌더링했다.
- `document.getElementById`, `createElement`, `appendChild`로 DOM 요소를 읽고 추가했다.
- URLSearchParams, DOMContentLoaded, GET/POST 전송 방식 등 웹 페이지 간 데이터 흐름을 다뤘다.

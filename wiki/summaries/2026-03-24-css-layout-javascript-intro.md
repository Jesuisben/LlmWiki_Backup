# 2026-03-24 CSS 레이아웃과 JavaScript 입문

## 한 줄 요약

CSS 박스 모델·position·글꼴 스타일과 JavaScript 기초를 배운 날이다.

## 배운 내용

- Element는 여는 태그와 닫는 태그, 내용까지 포함한 HTML 구성 단위다.
- 박스 모델, `position`, 여백, 리스트 스타일, 글꼴, 줄 간격, 하이라이트 등 CSS 표현을 실습했다.
- `script` 태그 안에서 JavaScript를 작성하고 변수, 배열, 객체를 다뤘다.
- `===`, `const`, `undefined` 같은 JavaScript 입문 포인트를 정리했다.

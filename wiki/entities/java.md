# Java

## 무엇인가

Java는 객체지향 프로그래밍 언어이며, 이 위키에서는 백엔드 개발과 Spring Boot 학습의 출발점이다.

## 이 위키에서의 맥락

사용자는 2026-02-26부터 Java를 배우기 시작했다. 첫날에는 IntelliJ에서 프로젝트·패키지·클래스를 만들고 출력문을 실행했으며, 이후 변수/자료형/연산자, 제어문, 배열, 클래스/객체, 상속, 추상 클래스, 인터페이스, `static`/`final`까지 이어졌다.

## 핵심 기능 / 특징

- 클래스 기반 객체지향 언어
- 기본 자료형과 참조 자료형 구분
- `if`, `switch`, `for`, `while` 제어문
- 배열과 객체 배열
- 캡슐화, 상속, 다형성, 추상화
- Spring Boot 백엔드 학습의 기반 언어

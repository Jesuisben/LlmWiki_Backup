# Folium

## 무엇인가
Folium은 위도·경도 데이터로 HTML 기반 대화형 지도를 만드는 Python 라이브러리다.

## 이 위키에서의 맥락
자전거 대여소·커피 매장 DataFrame을 지도에 표시하며 Marker, MarkerCluster, GeoJSON 경계, HeatMap을 사용했다. 생성한 지도는 HTML 파일로 저장해 브라우저에서 확인했다.

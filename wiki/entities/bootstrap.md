# Bootstrap

## 무엇인가

미리 만들어진 CSS class와 컴포넌트로 반응형 웹 UI를 빠르게 구성하는 프론트엔드 도구다.

## 이 위키에서의 맥락

- 2026-03-25에 CDN, grid system, table, card, form, button을 W3Schools 예제로 확인했다.
- 2026-03-26에는 상품 카드 목록 UI와 연결됐다.
- 2026-04-02에는 React Bootstrap의 Carousel/Card로 홈 화면을 구성했다.

## 핵심 기능 / 특징

- CDN으로 빠르게 불러오기
- container/row/col 기반 반응형 레이아웃
- btn, form-control, card, table 등 기본 UI class
- React Bootstrap 컴포넌트로 확장

## 헷갈리기 쉬운 점

Bootstrap은 HTML/CSS를 대체하지 않는다. 자주 쓰는 CSS 패턴을 class와 컴포넌트로 묶어 둔 도구다.

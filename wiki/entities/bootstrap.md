# Bootstrap

## 무엇인가

미리 만들어진 CSS class와 UI 구성 요소로 반응형 웹 화면을 빠르게 만드는 프론트엔드 프레임워크다.

## 이 위키에서의 맥락

- 2026-03-25에 CDN, grid system, table, form, button을 W3Schools와 교안으로 확인했다.
- `B.nobootstrap`와 `C.yesbootstrap` 소스 비교를 통해 직접 CSS 작성과 Bootstrap class 사용의 차이를 배웠다.
- 2026-03-26에는 상품 목록 카드와 상세 화면에서 `container`, `row`, `col-md-4`, `card`, `btn`을 사용했다.
- 2026-04-02에는 React Bootstrap의 Carousel/Card로 확장됐다.

## 핵심 기능 / 특징

- CDN으로 빠르게 불러오기
- `container/row/col` 기반 반응형 레이아웃
- `table`, `btn`, `form-control`, `form-select`, `card` 등 기본 UI class
- 직접 CSS를 대체하기보다 반복 패턴을 표준화

## 학습 이력

Bootstrap은 UI&UX 수업에서 직접 CSS를 작성한 뒤 “같은 화면을 미리 정의된 class로 더 빠르게 만드는 방식”으로 등장했다. 이후 React Bootstrap으로 넘어가며 컴포넌트 기반 UI 구성으로 연결된다.

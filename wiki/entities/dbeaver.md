# DBeaver

## 무엇인가

Oracle/MySQL 같은 데이터베이스에 접속해 SQL을 실행하고 테이블·스키마·ERD를 확인하는 데이터베이스 IDE다.

## 이 위키에서의 맥락

- 2026-03-16 Oracle 입문에서 SQL 실습 화면으로 사용했다.
- 행 번호 표시, SQL 실행, 결과 표 확인을 통해 행(row)과 열(column)을 눈으로 확인했다.
- 2026-03-20에는 ERD/엔티티 관계도 확인과 데이터 모델링 흐름에도 연결됐다.

## 핵심 기능 / 특징

- SQL 편집기에서 조회·정의·조작 명령 실행
- 결과를 표로 확인하며 데이터 상태 검증
- 제약조건·시퀀스 테스트 결과 확인
- ERD로 테이블 관계 파악

## 헷갈리기 쉬운 점

DBeaver는 데이터베이스 자체가 아니라 데이터베이스에 접속하는 작업 도구다.

# CSS

## 무엇인가

HTML 요소의 색, 크기, 배치, 글꼴, 여백, 애니메이션 같은 표현을 담당하는 스타일 언어다.

## 이 위키에서의 맥락

- 2026-03-23에 선택자, class/id, top-down 방식으로 시작했다.
- 2026-03-24에는 박스 모델, position, 여백, 리스트 스타일, 글꼴, 줄 간격을 실습했다.
- 2026-03-27에는 transition, transform, pseudo-class로 hover 효과를 다뤘다.

## 핵심 기능 / 특징

- 선택자로 스타일 적용 대상 지정
- 박스 모델과 position으로 배치 이해
- transition/transform/:hover로 간단한 시각 효과 구현
- Bootstrap class와 함께 사용

## 헷갈리기 쉬운 점

CSS는 작성 순서만이 아니라 선택자 우선순위와 구체성도 함께 작동한다.

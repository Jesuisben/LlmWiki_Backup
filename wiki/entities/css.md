# CSS

## 무엇인가

HTML 요소의 색, 크기, 배치, 글꼴, 여백, 테두리, 전환 효과 같은 표현을 담당하는 스타일 언어다.

## 이 위키에서의 맥락

- 2026-03-23에 inline style, 태그/class/id 선택자, top-down 적용 흐름으로 시작했다.
- 2026-03-24에는 Box Model, position, overflow, display, z-index, float, font를 실습했다.
- 2026-03-27에는 transition, transform, pseudo-class로 hover 확대와 이미지 효과를 만들었다.

## 핵심 기능 / 특징

- 선택자로 스타일 적용 대상 지정
- Box Model과 position으로 배치 이해
- `:hover`, `transition`, `transform`으로 UI 반응 표현
- Bootstrap class와 jQuery class 조작의 기반

## 학습 이력

CSS는 처음에는 글자색과 목록 스타일을 바꾸는 도구였지만, 곧 Box Model과 배치, Bootstrap class, jQuery의 class 토글까지 이어졌다. 즉 “화면 상태를 어떻게 보이게 할 것인가”의 공통 기반이다.

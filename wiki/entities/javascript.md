# JavaScript

## 무엇인가

브라우저에서 웹 페이지의 동작과 상호작용을 구현하는 프로그래밍 언어다.

## 이 위키에서의 맥락

- 2026-03-24에 `<script>`, 변수, 객체, 배열, `undefined`, `==`/`===`로 시작했다.
- 2026-03-26에는 상품 배열을 Bootstrap 카드 목록으로 동적 렌더링했다.
- `ProductList.html`에서는 `forEach`, template string, `createElement`, `appendChild`, `onclick`, `confirm`을 사용했다.
- `ProductDetail.html`에서는 loading/error/success 화면을 `style.display`로 전환했다.
- 이후 jQuery와 React/TypeScript 학습의 기반이 됐다.

## 핵심 기능 / 특징

- 배열·객체로 화면 데이터 표현
- DOM 요소 선택·생성·삽입
- 이벤트 기반 프로그래밍
- URL parameter와 페이지 이동 처리

## 학습 이력

JavaScript는 HTML/CSS로 만든 정적인 화면에 “조건, 반복, 이벤트, 페이지 이동”을 붙이는 단계로 등장했다. 이 사고방식은 React에서 state와 렌더링으로 이어진다.

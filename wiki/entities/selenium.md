# Selenium

## 무엇인가
Selenium은 실제 브라우저를 자동화해 클릭·페이지 이동·대기 같은 동적 웹 동작을 수행하는 도구다.

## 이 위키에서의 맥락
Python 수업에서 커피 매장 찾기 화면을 WebDriver로 열고, CSS selector와 `WebDriverWait`로 서울·전체 선택을 수행한 뒤 `page_source`를 파싱하는 실습에 사용했다.

# jQuery

## 무엇인가

JavaScript DOM 선택, 이벤트 처리, class/속성 조작을 짧게 작성하게 도와주는 라이브러리다.

## 이 위키에서의 맥락

- 2026-03-27에 JavaScript의 라이브러리로 등장했다.
- Library vs Framework 비교에서 jQuery는 “필요할 때 호출해 쓰는 도구” 예시로 다뤄졌다.
- `jQueryImageTest.html`에서 ready, click/change 이벤트, attr, class 토글, show/hide, append/prepend, chaining을 실습했다.

## 핵심 기능 / 특징

- `$()` 선택자로 DOM 요소 선택
- `click`, `change` 이벤트 처리
- `addClass`, `removeClass`, `toggleClass`로 CSS class 변경
- `attr`, `show`, `hide`, `append`, `prepend`와 method chaining

## 학습 이력

jQuery는 순수 JavaScript DOM 조작을 배운 다음날, 같은 작업을 더 짧고 선택자 중심으로 쓰는 방법으로 등장했다. React 이후에는 사용 빈도가 줄 수 있지만, DOM 선택과 이벤트의 기본 감각을 익히는 데 중요한 중간 단계다.

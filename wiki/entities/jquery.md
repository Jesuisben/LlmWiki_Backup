# jQuery

## 무엇인가

JavaScript DOM 선택, 이벤트 처리, class/속성 조작을 짧게 작성하게 도와주는 라이브러리다.

## 이 위키에서의 맥락

- 2026-03-27에 JavaScript의 라이브러리로 등장했다.
- CDN 로딩, `$()` 선택자, ready, click/change 이벤트, attr, toggleClass, show/hide, method chaining을 실습했다.
- 순수 JavaScript DOM 조작 이후 더 간결한 UI 상호작용 방식으로 배웠다.

## 핵심 기능 / 특징

- `$()` 선택자로 DOM 요소 선택
- click/change 이벤트 처리
- addClass/removeClass/toggleClass로 CSS class 변경
- attr/show/hide와 method chaining

## 헷갈리기 쉬운 점

jQuery는 JavaScript와 별개의 언어가 아니라 JavaScript로 만든 라이브러리다.

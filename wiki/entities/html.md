# HTML

## 무엇인가

웹 페이지의 구조와 의미를 표현하는 마크업 언어다.

## 이 위키에서의 맥락

- 2026-03-23 UI&UX 수업에서 태그, 속성, 문자 엔티티, 색상, 목록, 표, 이미지 대체 텍스트를 배웠다.
- 2026-03-25에는 form, select, button, placeholder 같은 입력 요소로 확장됐다.
- 이후 React JSX에서 HTML과 비슷한 구조를 다시 사용한다.

## 핵심 기능 / 특징

- 태그와 속성으로 문서 구조 표현
- 표·이미지·폼·버튼 같은 UI 기본 요소 구성
- CSS와 JavaScript가 붙을 기준점 제공
- React 컴포넌트 마크업의 기반

## 헷갈리기 쉬운 점

HTML은 디자인 언어가 아니라 구조 언어다. 모양은 CSS, 동작은 JavaScript가 담당한다.

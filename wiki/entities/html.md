# HTML

## 무엇인가

웹 페이지의 구조와 의미를 표현하는 마크업 언어다. 수업 자료에서는 Hyper Text Markup Language로 소개했고, tag/element/attribute/value를 기본 용어로 다뤘다.

## 이 위키에서의 맥락

- [[summaries/2026-03-23-html-css-intro|2026-03-23]]에 UI&UX 주간의 첫 도구로 등장했다.
- `tableExam.html`에서 표, 이미지, 링크, `alt`를 함께 실습했다.
- [[summaries/2026-03-25-bootstrap-form|2026-03-25]]에는 form, select, button, placeholder 같은 입력 요소로 확장됐다.
- [[summaries/2026-03-27-uiux-subject-review|UI&UX 총정리]]는 HTML/CSS/JavaScript/Bootstrap을 한 번에 복습하는 허브다.
- 이후 React JSX에서도 HTML과 비슷한 구조를 계속 사용한다.

## 핵심 기능 / 특징

- 태그와 속성으로 문서 구조 표현
- 표·이미지·폼·버튼 같은 UI 기본 요소 구성
- CSS와 JavaScript가 붙을 기준점 제공
- URL parameter, form submit, DOM 선택의 출발점

## 학습 이력

HTML은 Java/Oracle 이후 “데이터를 사용자에게 보여주는 층”을 만들기 위한 첫 프론트엔드 학습이었다. 구조를 HTML로 만들고, 표현은 CSS/Bootstrap, 동작은 JavaScript/jQuery로 확장했다.

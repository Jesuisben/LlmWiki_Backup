# Python 기본 문법

## 정의

Python 기본 문법은 값을 저장하고, 출력하고, 입력받고, 조건과 반복으로 실행 흐름을 제어하는 최소 문법 묶음이다.

## 왜 중요한가

Pandas와 데이터 분석 실습은 이 기초 위에서 진행된다. 사용자가 Java, Oracle, Spring/React를 먼저 배웠기 때문에 Python 개념은 기존의 자료형, 컬렉션, 객체지향, 파일 처리, SQL 테이블 감각과 연결해 이해하는 것이 좋다.

## 핵심 설명

`print`, `input`, 형변환, 문자열 함수, `if/elif/else`, `for`, `while`, `range`는 Pandas 코드에서도 계속 사용된다. 특히 `input()`은 문자열을 반환하고, `range`의 끝값은 포함되지 않으며, Python 블록은 들여쓰기로 결정된다.

## 예시

2026-06-19의 성적 입력·문자열 처리 실습과 2026-06-22의 학점 판정·합계·커피 판매 실습이 기초 문법의 대표 예시다.

## 자주 헷갈리는 점

- Python은 타입을 변수 선언에 적지 않고 값에 따라 결정한다.
- 짧은 문법일수록 결과 타입이 무엇인지 확인하는 습관이 필요하다.
- 순수 Python 자료구조와 Pandas DataFrame은 목적은 비슷해도 동작 방식과 제공 함수가 다르다.
